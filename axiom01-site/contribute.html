Contribute to Axiom01

Axiom01 is open source and welcomes contributions from everyone. Help us build the next great UI framework!

How to Contribute

  1. Read our Style Guide and Documentation.
  2. Fork the repository on GitHub and submit pull requests.
  3. Report issues and suggest features via GitHub Issues or Discord.
  4. Join discussions and share feedback in our community chat.

Example Contribution

Want to add a new component? Fork, create your feature branch, and submit a PR with a demo and documentation update.

Community

Connect with other users and contributors. Share your projects and ideas!

  • GitHub Discussions
  • Discord Community
  • Twitter

Code of Conduct

We value respectful, inclusive collaboration. Please review our code of conduct before participating.
  • Be kind and constructive.
  • Respect all contributors and users.
  • Report violations to maintainers.
