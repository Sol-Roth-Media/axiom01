Getting Started

Install Axiom01 via CDN or npm, and include axiom.css in your project.

<link rel="stylesheet" href="/css/axiom.css">
npm install axiom01

Import in your CSS or JS:

@import "axiom01/css/axiom.css";

Philosophy

Axiom01 is built for clarity, accessibility, and rapid development. It embraces semantic HTML, modular CSS, and open contribution. Our goal: make beautiful, accessible interfaces easy for everyone.

  • Semantic-first: Use HTML as intended
  • Accessible by default
  • Customizable and extensible
  • Open source and community-driven

Theming & Variables

Customize colors, spacing, and typography using CSS variables in axiom_vars.css:

:root {
  --axiom-primary: #6366f1;
  --axiom-success: #22c55e;
  --axiom-radius: 1rem;
  /* ...more variables... */
}

Override variables in your own CSS to match your brand.

Best Practices

  • Use semantic HTML elements.
  • Leverage utility classes for layout and spacing.
  • Test for accessibility and responsiveness.
  • Contribute improvements and report issues.
