Axiom01 CSS Variables & Properties API

A comprehensive reference for all CSS variables and custom properties used in Axiom01. Use these tokens to customize themes, layouts, colors, spacing, and more.

Color Variables

:root {
  --a-color-primary: #007bff;
  --a-color-success: #28a745;
  --a-color-warning: #ffc107;
  --a-color-danger: #dc3545;
  --a-color-info: #17a2b8;
  --a-color-surface: #fff;
  --a-color-surface-variant: #f4f4f4;
  --a-color-background: #fff;
  --a-color-on-surface: #222;
  --a-color-on-background: #222;
  --a-color-muted: #888;
}

Override these in your theme or use data-theme for dynamic switching.

:root {
  --a-padding-base: 1rem;

  --a-padding-xlarge: 3rem;
  --a-margin-base: 1rem;
  --a-margin-large: 2rem;

:root {
  --a-grid-gutter: 1.5rem;

}

Component-Specific Variables

:root {
  --a-btn-padding: 0.75rem 1.5rem;
  --a-btn-radius: 6px;
  --a-card-padding: 1.5rem;
  --a-card-radius: 8px;
  --a-alert-padding: 1rem 2rem;
  --a-alert-radius: 6px;
}

Usage Examples

/* Override a variable in your CSS */
:root {
  --a-color-primary: #e91e63;
}

/* Use a variable in a component */
.button {
  background: var(--a-color-primary);
  padding: var(--a-btn-padding);
  border-radius: var(--a-btn-radius);
}

Custom Properties API

All variables are available for use in your own CSS. Reference them with var(--variable-name) anywhere in your stylesheets.

  • Colors: var(--a-color-primary), var(--a-color-success), etc.
  • Spacing: var(--a-padding-base), var(--a-margin-large)
  • Typography: var(--a-font-size-h1), var(--a-font-family-sans)
  • Grid: var(--a-grid-gutter), var(--a-grid-columns)
  • Components: var(--a-btn-padding), var(--a-card-radius)
