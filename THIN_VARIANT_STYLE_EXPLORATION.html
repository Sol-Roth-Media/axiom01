Detailed Style Recommendations

🎯 RECOMMENDED: Outlined + Lighter (Option 2B)

★★★★★

Approach: Keep 1px strokes + add 60-70% opacity

<g stroke-width="1" opacity="0.65"> <path d="..."/> <circle.../> </g>

Why this works:

  • ✓ Immediately visually distinct
  • ✓ Elegant and refined appearance
  • ✓ Works at any size
  • ✓ Minimal file size (just opacity attr)
  • ✓ Easy to adjust (change opacity value)
  • ✓ Professional look (like Material Design's "outlined" style)

Implementation: Single attribute per Thin variant.

ALTERNATIVE: Truly Outlined (0.75-1px)

★★★★☆

Approach: Use 0.75px or 1px but scaled down shapes

<g stroke-width="0.75" fill="none"> <path d="..."/> <circle.../> </g>

Pros & Cons:

  • ✓ Ultimate visual distinction
  • ✓ "Paper-thin" elegant look
  • ✗ May look too faint at small sizes
  • ✗ Anti-aliasing issues on some displays
  • ✗ Less readable in dense layouts

Best for: Large icon displays (32px+)

My Recommendation: Go with Option 2B (Outlined + Lighter with 65% opacity) because:
  • Immediately creates visual hierarchy (dark regular, light thin)
  • Works perfectly at any DPI or screen size
  • Easy to implement (just add opacity="0.65" to Thin wrapper)
  • Can be adjusted later if needed (try 0.5-0.7 range)
  • Matches popular design systems (Material, Ant Design)
  • Minimal code/file size impact
