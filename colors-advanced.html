How Theming Works

Axiom's theming is built entirely on CSS custom properties (variables). We define a set of semantic color variables that are used throughout the framework's components. To create a new theme, you simply need to provide new values for these variables within a `[data-theme="your-theme-name"]` attribute selector.

Core Color Variables

These are the essential variables you need to define for a new theme. All other colors can be derived from these.

  • --a-color-primary: The main brand color.
  • --a-color-on-primary: Text color for on top of primary.
  • --a-color-secondary: The secondary brand color.
  • --a-color-on-secondary: Text color for on top of secondary.
  • --a-color-surface: The main background color.
  • --a-color-on-surface: The main text color.
  • --a-color-outline: Border and divider color.

Creating a Custom Theme

Here is the complete code for the "Forest" theme. You can use this as a template for your own themes.

[data-theme="forest"] {
    --a-color-primary: #4CAF50;
    --a-color-on-primary: #FFFFFF;
    --a-color-secondary: #81C784;
    --a-color-on-secondary: #000000;
    --a-color-surface: #E8F5E9;
    --a-color-on-surface: #1B5E20;
    --a-color-outline: #A5D6A7;
}
