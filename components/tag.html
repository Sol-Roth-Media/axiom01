Tag Component

Overview

The Tag component displays small, contextual labels, categories, or keywords. Tags are semantic, accessible, and use minimal classes. Removable tags use a button for interaction and accessibility.

Usage

<span class="tag">Example Tag</span>
<span class="tag primary">Primary Tag</span>
<span class="tag success">Success Tag</span>
<span class="tag removable">Removable Tag <button aria-label="Remove tag">×</button></span>

Demo

Options

  • tag primary, tag success for color variants
  • tag removable for removable tags

Accessibility

  • Removable tags use a button with aria-label for screen readers.
  • Tags use semantic <span> and <button> elements only.

Customization

  • Style with semantic CSS variables: --a-color-primary, --a-color-success, --a-color-surface-variant, etc.
  • Adjust border radius, font size, and spacing with CSS variables.
