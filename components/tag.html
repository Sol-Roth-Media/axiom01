Tag Component

The Tag component is used to display small, contextual labels, categories, or keywords. Tags can be styled, removable, and interactive.

Usage

<span class="tag">Example Tag</span>
<span class="tag removable">Removable Tag <button aria-label="Remove tag">×</button></span>

Demo

Default Tag Primary Tag Success Tag Removable Tag

Options

  • tag primary, tag success, etc. for color variants
  • tag removable for removable tags

Accessibility

  • Removable tags use a button with aria-label for screen readers.

Customization

  • Style with semantic CSS variables: --a-color-primary, --a-color-success, etc.
