Notification Component

Overview

The Notification component displays important messages to users, such as alerts, confirmations, and status updates. It is accessible, themeable, and uses semantic markup with axiom01 styling.

Example

Basic Notification: Use notifications to inform users about important events, confirmations, errors, or warnings. Notifications should be clear, concise, and accessible. The following examples show success and error notifications with icons and dismiss buttons.

✔ Success! Your changes have been saved.
⚠ Error! Something went wrong. 
<div class="alert" role="alert" aria-live="polite" style="background:var(--a-color-success);color:#fff;padding:1rem 2rem;border-radius:6px;min-width:180px;">
  <span class="alert-icon" aria-hidden="true">✔</span>
  Success! Your changes have been saved.
  <button class="closeBtn" aria-label="Close notification" onclick="this.parentElement.style.display='none'">&times;</button>
</div>
<div class="alert" role="alert" aria-live="assertive" style="background:var(--a-color-error);color:#fff;padding:1rem 2rem;border-radius:6px;min-width:180px;margin-top:1rem;">
  <span class="alert-icon" aria-hidden="true">⚠</span>
  Error! Something went wrong.
  <button class="closeBtn" aria-label="Close notification" onclick="this.parentElement.style.display='none'">&times;</button>
</div>

Classes, Attributes, and Options

  • .alert: Base notification style
  • .alert-icon: Optional icon for context
  • .closeBtn: Dismiss button, accessible
  • role="alert": Ensures screen readers announce the message
  • aria-live: Controls announcement urgency (polite or assertive)

Accessibility Features

  • Uses role="alert" and aria-live for screen reader support
  • Dismiss button is keyboard accessible and labeled
  • Color contrast meets WCAG standards in all themes

Customization Tips

  • Change colors using CSS variables: --a-color-success, --a-color-error, --a-color-warning, --a-color-info
  • Adjust spacing with --a-padding-base, --a-margin-base
  • Modify border radius with --a-border-radius-base
