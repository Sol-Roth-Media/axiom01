Alert Component Demo

Overview

The Alert component displays important messages to users, such as confirmations, warnings, errors, and informational notes. Alerts use semantic color states and can include icons, links, and dismiss buttons for accessibility and usability.

Examples

Basic Alerts: Use alerts to communicate status or feedback. Each alert uses a semantic color and role="alert" for accessibility.

This is a success alert!
This is a warning alert!
This is an error alert!
This is an info alert! 
<section class="alert success" role="alert">
  This is a success alert!
</section>
<section class="alert error" role="alert">
  This is an error alert!
</section>

Advanced Alerts: Add a close button for dismissible alerts, icons for context, and extra content such as links or bold text.

✔ This is a success alert!
⚠ This is a warning alert!
✖ This is an error alert!
ℹ This is an info alert!
Heads up! This is an advanced info alert with a close button and extra content.
Learn more 
<section class="alert success" role="alert">
  <span class="icon" aria-hidden="true">✔</span>
  This is a success alert!
  <button class="close" aria-label="Close" onclick="this.parentElement.style.display='none'">×</button>
</section>
<section class="alert info" role="alert">
  <strong>Heads up!</strong> This is an advanced info alert with a close button and extra content.<br>
  <a href="#" style="color:inherit;text-decoration:underline">Learn more</a>
  <button class="close" aria-label="Close" onclick="this.parentElement.style.display='none'">×</button>
</section>

How to use the Alert Component

Usage: Place alerts anywhere you need to communicate status, feedback, or important information. Use semantic color classes for context, and add icons or close buttons for clarity and interactivity.
Accessibility: Always use role="alert" for screen reader support, and label close buttons with aria-label. Icons should use aria-hidden="true" so they are not announced by screen readers. 

<!-- Success alert -->
<section class="alert success" role="alert">
  <span class="icon" aria-hidden="true">✔</span>
  Operation successful!
</section>

<!-- Error alert with close button -->
<section class="alert error" role="alert">
  <span class="icon" aria-hidden="true">✖</span>
  Something went wrong.
  <button class="close" aria-label="Close" onclick="this.parentElement.style.display='none'">×</button>
</section>

<!-- Info alert with extra content -->
<section class="alert info" role="alert">
  <strong>Heads up!</strong> This is an info alert with a link and close button.<br>
  <a href="#" style="color:inherit;text-decoration:underline">Learn more</a>
  <button class="close" aria-label="Close" onclick="this.parentElement.style.display='none'">×</button>
</section>

Options

  • Type: success, warning, error, info
  • Icon: Use <span class="icon"> for a semantic icon
  • Close Button: Add <button class="close"> for dismissible alerts
  • Extra Content: Add links, bold text, or other elements as needed
  • Accessibility: Always use role="alert" and aria-label for close buttons
