Infinite Scroll Component

Overview

The Infinite Scroll component loads more content as the user scrolls, providing a seamless browsing experience. It is accessible, themeable, and uses semantic markup with axiom01 styling.

Classes, Attributes, and Options

  • #infinite-scroll-demo: Scrollable container
  • #scroll-list: List of items
  • #loading-indicator: Loading feedback

Accessibility Features

  • Keyboard and screen reader accessible
  • Loading indicator is visible and announced
  • Color contrast meets WCAG standards in all themes

Customization Tips

  • Change container and item styles using CSS variables: --a-color-surface, --a-color-surface-variant
  • Adjust spacing with --a-padding-base, --a-margin-base
  • Modify border radius with --a-border-radius-base
