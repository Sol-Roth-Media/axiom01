Navbar Component

The Navbar provides site navigation and supports dropdowns, jump menus, and responsive layouts.

Usage

<nav class="navbar">
  <ul>
    <li><a href="#">Home</a></li>
    <li class="dropdown">
      <a href="#" aria-haspopup="true" aria-expanded="false">More</a>
      <ul aria-label="submenu">
        <li><a href="#">Docs</a></li>
      </ul>
    </li>
    <li>
      <select onchange="location = this.value;" aria-label="Jump Menu">
        <option value="#">Jump to...</option>
      </select>
    </li>
  </ul>
</nav>

JavaScript Example

document.addEventListener('DOMContentLoaded', () => {
  document.querySelectorAll('.navbar .dropdown > a').forEach(toggle => {
    toggle.addEventListener('click', function(e) {
      e.preventDefault();
      const menu = this.nextElementSibling;
      const expanded = this.getAttribute('aria-expanded') === 'true';
      this.setAttribute('aria-expanded', !expanded);
      menu.hidden = expanded;
    });
  });
});

Accessibility

  • Dropdowns use aria-haspopup and aria-expanded.
  • Jump menu uses aria-label for screen readers.
  • Keyboard and screen reader accessible.
