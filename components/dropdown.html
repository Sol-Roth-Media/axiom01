Dropdown Menu Component

The Dropdown Menu provides a list of options that can be shown or hidden. It supports keyboard navigation and ARIA attributes for accessibility.

Usage

<div class="dropdown">
  <button aria-haspopup="true" aria-expanded="false">Menu</button>
  <ul hidden>
    <li><a href="#">Item</a></li>
  </ul>
</div>

JavaScript Example

document.addEventListener('DOMContentLoaded', () => {
  const dropdown = document.querySelector('.dropdown');
  const toggle = dropdown.querySelector('button');
  const menu = dropdown.querySelector('ul');
  toggle.addEventListener('click', () => {
    const expanded = toggle.getAttribute('aria-expanded') === 'true';
    toggle.setAttribute('aria-expanded', !expanded);
    menu.hidden = expanded;
  });
});

Accessibility

  • Uses aria-haspopup, aria-expanded, and aria-labelledby.
  • Keyboard and screen reader accessible.
