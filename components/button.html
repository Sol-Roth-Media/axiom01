Button Component Demo

Buttons are used for actions and navigation. Use semantic classes for color and style. Below are basic and advanced examples.

Basic Buttons

<button class="primary">Primary Button</button>
<button>Default Button</button>
<button class="danger">Danger Button</button>

Advanced Buttons

<button class="primary" disabled>Disabled Primary</button>
<button class="primary"><span class="icon" aria-hidden="true">★</span> With Icon</button>
<button class="danger"><span class="icon" aria-hidden="true">!</span> Danger with Icon</button>
<button class="primary" onclick="alert('Clicked!')">Action Button</button>

Usage

  • Primary: class="primary" for main actions
  • Danger: class="danger" for destructive actions
  • Default: no class for neutral actions
  • Disabled: disabled attribute for inactive buttons
  • Icons: <span class="icon"> for inline icons
  • Actions: onclick for interactive behavior
