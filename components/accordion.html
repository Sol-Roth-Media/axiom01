Accordion Component

The Accordion component allows users to expand and collapse sections of content. It is fully accessible, using ARIA attributes and keyboard navigation.

Usage

Use semantic HTML for structure. The button toggles the visibility of its associated panel.

<div class="accordion">
  <div>
    <h3>
      <button type="button" aria-expanded="false" aria-controls="panel-1" id="trigger-1">
        Section 1
      </button>
    </h3>
    <div id="panel-1" role="region" aria-labelledby="trigger-1" hidden>
      <p>Content for Section 1.</p>
    </div>
  </div>
</div>

JavaScript Initialization

Accordion functionality is enabled by the following JavaScript (included in scripts.js):

document.addEventListener('DOMContentLoaded', () => {
  document.querySelectorAll('.accordion button').forEach(trigger => {
    trigger.addEventListener('click', () => {
      const panel = document.getElementById(trigger.getAttribute('aria-controls'));
      const expanded = trigger.getAttribute('aria-expanded') === 'true';
      trigger.setAttribute('aria-expanded', !expanded);
      panel.toggleAttribute('hidden');
    });
  });
});

Accessibility

  • Uses aria-expanded and aria-controls for state and association.
  • Panels use role="region" and aria-labelledby for screen readers.
  • Keyboard navigation is supported.
