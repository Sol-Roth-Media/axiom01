Progress Throbber Component

The Progress Throbber component displays a spinning indicator to show loading or ongoing activity. It is customizable for size, color, and animation speed.

Usage

<span class="axiom-throbber" aria-label="Loading" role="status"></span>
<span class="axiom-throbber axiom-throbber--large" aria-label="Loading" role="status"></span>
<span class="axiom-throbber axiom-throbber--primary" aria-label="Loading" role="status"></span>

Demo

Options

  • axiom-throbber--large, axiom-throbber--small for size
  • axiom-throbber--primary, axiom-throbber--success, etc. for color
  • axiom-throbber--fast, axiom-throbber--slow for animation speed
