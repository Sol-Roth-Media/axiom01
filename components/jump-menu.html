Jump Menu Component

The Jump Menu provides a quick way for users to navigate to different sections or pages. It is accessible and easy to implement.

Demo

Usage

<select aria-label="Jump Menu" onchange="if(this.value) location=this.value;">
  <option value="">Select...</option>
  <option value="#section1">Section 1</option>
</select>

Accessibility

  • Uses aria-label for screen readers.
  • Keyboard and screen reader accessible.
