Datepicker Component

Overview

The Datepicker component allows users to select dates from a calendar interface. It is accessible, themeable, and uses semantic markup with axiom01 styling.

Example

Classes, Attributes, and Options

  • .datepicker: Base datepicker style
  • type="date": Native HTML date input
  • aria-label: Add for custom accessibility labeling

Accessibility Features

  • Uses native HTML input for full keyboard and screen reader support
  • Label is associated with input for clarity
  • Color contrast meets WCAG standards in all themes

Customization Tips

  • Change border and background using CSS variables: --a-color-surface, --a-color-surface-variant
  • Adjust spacing with --a-padding-base, --a-margin-base
  • Modify border radius with --a-border-radius-base
