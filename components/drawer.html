Mobile Navigation Drawer

The mobile navigation drawer provides a slide-in menu for mobile and tablet devices. It is themeable, accessible, and easy to use with Axiom01.

How to Use: Drawer

<button id="drawer-toggle">☰</button>
<aside id="drawer">
  <nav>
    <ul>...</ul>
  </nav>
</aside>
  • Use <aside> for the drawer and <nav> for navigation.
  • Accessible with aria-label and semantic HTML.
  • Theme with CSS variables and transitions.
