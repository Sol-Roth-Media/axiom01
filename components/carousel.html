Carousel Component

Overview

The Carousel component displays a series of images or content slides, allowing users to navigate between them. It is accessible, themeable, and uses semantic markup with axiom01 styling.

Classes, Attributes, and Options

  • .carousel: Base carousel container
  • .carousel-track: Slide track (flex container)
  • .carousel-slide: Individual slide
  • .carousel-controls: Navigation controls
  • role="region": Accessibility region
  • aria-label: Describes carousel for screen readers

Accessibility Features

  • Keyboard navigation via tab and arrow keys
  • Screen reader region labeling
  • Color contrast meets WCAG standards in all themes

Customization Tips

  • Change slide background and border using CSS variables: --a-color-surface, --a-border-radius-base
  • Adjust spacing with --a-padding-base, --a-margin-base
  • Modify controls with .btn classes and theme variables
