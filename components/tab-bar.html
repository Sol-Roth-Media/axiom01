Tab Bar Component

The Tab Bar component provides a horizontal navigation bar for switching between different views or sections. Tabs are accessible, keyboard-navigable, and can be styled for active/inactive states.

Usage

<nav class="axiom-tab-bar" role="tablist">
  <button class="axiom-tab" role="tab" aria-selected="true">Tab 1</button>
  <button class="axiom-tab" role="tab">Tab 2</button>
  <button class="axiom-tab" role="tab">Tab 3</button>
</nav>

Demo

Options

  • Use aria-selected="true" for the active tab.
  • Tabs can be styled with axiom-tab--primary, axiom-tab--accent, etc.

Accessibility

  • Tabs use role="tablist" and role="tab" for screen readers.
  • Keyboard navigation: left/right arrows to switch tabs.

Customization

  • Style with CSS variables: --a-tab-bg, --a-tab-color, etc.
