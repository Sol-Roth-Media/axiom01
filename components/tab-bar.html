Tab Bar Component

Overview

The Tab Bar component provides a horizontal navigation bar for switching between different views or sections. Tabs are accessible, keyboard-navigable, and can be styled for active/inactive states. Use semantic <nav> and <button> elements for structure, and minimal classes for styling. Always ensure tabs have clear labels and are easy to navigate.

Usage

<nav class="tab-bar" role="tablist">
  <button class="tab" role="tab" aria-selected="true" tabindex="0">Tab 1</button>
  <button class="tab" role="tab" tabindex="-1">Tab 2</button>
  <button class="tab" role="tab" tabindex="-1">Tab 3</button>
</nav>

Demo

Options

  • Use aria-selected="true" for the active tab.
  • Tabs can be styled with tab primary, tab accent, etc.

Accessibility

  • Tabs use role="tablist" and role="tab" for screen readers.
  • Keyboard navigation: left/right arrows to switch tabs.
  • Only the active tab should have tabindex="0"; others use -1.

Customization

  • Style with CSS variables: --a-tab-bg, --a-tab-color, etc.
