A Framework with a Philosophy

Axiom01 is not just another UI framework. It's a statement against the complexity and bloat that has taken over modern web development. We believe that a framework should serve you, not the other way around. Our philosophy is simple: write clean, semantic HTML, and let the framework do the rest.

While other frameworks encourage a sea of utility classes and `

` soup, Axiom champions a return to the principles of the semantic web. This approach leads to code that is more readable, more maintainable, and more accessible by default. It's a long-term investment in the health and sanity of your codebase.

Semantic First

Write clean, meaningful HTML. We'll handle the styling.

Minimal Classes

One class per component. No more, no less.

Accessible by Default

WCAG 2.1 AA compliance out of the box.

The Cascading Spacing System

Axiom's layout is built on a simple, powerful idea: a single source of truth for all spacing. The CSS variable --a-space-unit is the foundation for all sizing related to space. It generates a scale of sizes (from --a-space-xs to --a-space-xxl) used for padding, margin, and gap properties throughout the framework. Adjust the slider below to see how changing this one value can globally and consistently rescale the layout of an entire application.

Card (padding: --a-space-l)
Button (padding: --a-space-m)
Tag (padding: --a-space-s)

The container above uses gap: var(--a-space-m). Each item's margin and padding are derived from the same unit.

A Rich and Flexible Color System

Axiom's color system is designed for clarity, consistency, and adaptability. We provide a full spectrum of colors, from base brand colors to a complete greyscale palette and semantic colors for UI states. All colors are available as CSS variables for easy use and customization. For a deeper dive, visit the Advanced Color Details page or use the Theme Wizard.

Get Started in Seconds

Axiom is a free and open-source framework, released under the MIT License. You can use it for any personal or commercial project. Start building your next project today with one of the options below.

Use the CDN

The quickest way to get started. Just add the following stylesheet link to the <head> of your HTML file.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Sol-Roth-Media/axiom/css/core.css">

Download Locally

Download the full framework source code to host it yourself and get access to all component files and themes.

Why Choose Axiom?

Axiom is built on a different philosophy from other frameworks. We believe in clean, semantic HTML and a framework that gets out of your way. Here's how we compare to some of the most popular alternatives:

Feature Axiom Bootstrap Tailwind CSS
Semantic HTML Encouraged Discouraged Not Applicable
Class Usage Minimal Component-based Utility-first
Accessibility High High Developer-dependent
Customization High (via CSS Variables) Medium (via Sass) High (via config)
