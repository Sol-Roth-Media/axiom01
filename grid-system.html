Axiom01 Grid System

Axiom01 provides a flexible, responsive grid system using CSS Grid and Flexbox. Easily create layouts that adapt to any screen size with semantic HTML and minimal classes.

Grid Basics

The grid system uses display: grid and display: flex utilities. Example:

<div style="display:grid;grid-template-columns:repeat(3,1fr);gap:1rem;">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</div>

Responsive Grids

Use minmax and auto-fit for responsive columns:

<div style="display:grid;grid-template-columns:repeat(auto-fit,minmax(180px,1fr));gap:1rem;">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
  <div>Item 4</div>
</div>

Flexbox Utilities

Use display: flex for horizontal or vertical layouts:

<div style="display:flex;gap:1rem;">
  <div>Flex 1</div>
  <div>Flex 2</div>
  <div>Flex 3</div>

Advanced Grid Features

Combine grid and flex for complex layouts. Use named areas, row/column spans, and more.

/* Example: Grid with row/column spans */
<div style="display:grid;grid-template-columns:repeat(4,1fr);gap:1rem;">
  <div style="grid-column:span 2;">Wide</div>
  <div>A</div>
  <div>B</div>
  <div style="grid-row:span 2;">Tall</div>
  <div>C</div>
  <div>D</div>
</div>

Grid Design Tokens

Customize grid gaps, columns, and breakpoints using CSS variables in axiom_vars.css:

:root {
  --a-grid-gutter: 1.5rem;
  --a-grid-columns: 12;
  --a-grid-breakpoint-md: 900px;
}

Responsive Design Patterns

Below are examples of responsive grid layouts that collapse columns and adapt to different screen sizes using CSS Grid and media queries.

Collapsing Columns (2 to 1)

<style>
  .axiom-grid-collapse {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 1rem;
  }
  @media (max-width: 600px) {
    .axiom-grid-collapse {
      grid-template-columns: 1fr;
    }
  }
</style>
<div class="axiom-grid-collapse">
  <div>Column 1</div>
  <div>Column 2</div>
</div>

Multi-breakpoint Responsive Grid
