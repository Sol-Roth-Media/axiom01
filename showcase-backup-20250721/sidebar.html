Sidebar Navigation

A sidebar navigation provides persistent access to key sections of your site or app. It is themeable, accessible, and responsive with Axiom01.

How to Use: Sidebar Navigation

<aside>
  <nav>
    <ul>
      <li><a href="/">Home</a></li>
      ...
    </ul>
  </nav>
</aside>
  • Use <aside> for sidebar and <nav> for navigation.
  • Theme with CSS variables and flexbox for layout.
  • Accessible with aria-label and semantic HTML.
