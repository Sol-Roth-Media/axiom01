RELEASE STEPS
═══════════════════════════════════════════════════════════════════════════════

1. Verify Build
   $ npm run build
   Expected: Minified CSS, minified JS, dist artifacts built

2. Commit & Push
   $ git add -A
   $ git commit -m "Release v2.1.2: GitHub Pages fix, minified assets"
   $ git push origin main

3. Deploy to GitHub Pages
   $ git push origin gh-pages
   Wait 5-10 minutes, then verify at https://axiom01.com

4. Create GitHub Release (Optional)
   Tag: v2.1.2
   Title: v2.1.2 - Build Update and GitHub Pages Fix
   Description: Copy from CHANGELOG.md

5. Publish to NPM
   $ npm login
   $ npm publish

6. Verify Publication
   $ npm info axiom01
   Check: CDN and axiom01.com
