Axiom01 Logo Explorations

Three distinct directions based on your preference for a clean wordmark, stacked layout, or something completely different.

Option 1: Wordmark

Axiom

Clean, professional, memorable. Just "Axiom" in a bold monospace typeface with gradient accent. Works great at all sizes, scales beautifully in favicons. The gradient adds sophistication without complexity.

Uses: Nav logo, footer, favicons, docs

Option 2: Stacked

Axiom

Icon above text. Strong vertical rhythm. The A with ticks as a focal point, then "Axiom" below. Creates rhythm and hierarchy. Icon-first emphasis on the brand mark itself.

Uses: Landing pages, large displays, brand assets

Option 3: Modern Badge

Modern, app-like, distinctive. A with ticks contained in a sleek rounded-square badge. Contemporary feel, works as a favicon/app icon. The frame makes it feel like a branded app.

Uses: App icon, favicon, badge contexts

Usage Recommendations

Option 1 Works Best For:

  • Primary logo across all contexts
  • Header/footer branding
  • Favicons and small sizes
  • Professional documents
  • Minimalist approach

Option 2 Works Best For:

  • Landing page hero
  • Poster/marketing materials
  • Large displays
  • Icon + text separation
  • Rhythm-focused design

Option 3 Works Best For:

  • App/favicon context
  • Modern tech feel
  • Contained/rounded aesthetic
  • Badge backgrounds
  • Social media avatars

Technical Implementation Notes

Current Implementation: A10M (icon + 10 + M text)

Option 1 (Wordmark): Single text element with gradient, CSS-only. Minimal markup, scales perfectly.

Option 2 (Stacked): SVG icon + "Axiom" text in flexbox column. Great for mobile navbar with proper spacing.

Option 3 (Badge): SVG icon in a framed container. App-like appearance, strong visual boundary.

Which direction resonates with you? We can also combine elements — for example, use Option 1 in the header and Option 2 on the homepage hero.
