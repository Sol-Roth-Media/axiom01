## How Axiom01 Now Demonstrates Its Philosophy

Every page shows by example:

✓ **Semantic HTML First**
```html
<nav role="navigation">
  <ul><li><a href="/">Home</a></li></ul>
</nav>

<main>
  <section class="hero">
    <h1>Title</h1>
  </section>
</main>
```

✓ **Minimal Classes**
```html
<article class="card">
  <header><h3>Title</h3></header>
  <main><p>Content</p></main>
  <footer><button class="primary">Action</button></footer>
</article>
```

✓ **CSS Variables Only**
```css
.component-example {
  background: var(--a-color-surface-variant);
  padding: var(--a-space-l);
  border: 1px solid var(--a-color-outline);
}
```
