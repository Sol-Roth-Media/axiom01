Three Core Principles

AXIOM01 v3 is built on three foundational principles that drive every design decision.

Semantic-First

Write meaningful HTML where elements describe WHAT they ARE, not what they look like.

Pure semantic structure. One ID per component. Zero BEM classes.

Mathematical Design

OKLCH color space for perceptually uniform colors that automatically adapt to dark mode.

Design tokens. Spacing scales. Predictable systems.

Framework Responsibility

The framework handles CSS. You write semantic HTML. No build tools required.

Zero dependencies. Static hosting. Ship faster.
