Dynamic Content Management

Learn how to programmatically manage dynamic content in Axiom01 components using JavaScript helpers.

Overview

Axiom01 provides powerful helper classes for managing dynamic content in common UI patterns. These utilities make it easy to add, remove, update, and manipulate component content at runtime.

The three main helper classes are:

  • DynamicListHelper - Manage lists and collections of items
  • DynamicPaginationHelper - Handle pagination dynamically
  • DynamicGridHelper - Manage grid/card layouts

DynamicListHelper

Manage lists and collections with add, remove, update, and filter operations.

Basic Usage

import { DynamicListHelper } from '../js/components/dynamic-content-helpers.js';

// Initialize the helper with a container selector
const list = new DynamicListHelper('.my-list');

// Add a single item
list.addItem({
    id: 'item-1',
    title: 'First Item',
    description: 'This is the first item'
});

// Add multiple items
list.addItems([
    { id: 'item-2', title: 'Second Item' },
    { id: 'item-3', title: 'Third Item' }
]);

    Remove Items

    // Remove by ID
list.removeItem('item-1');

// Clear all items
list.clear();

    Update Items

    // Update item data and re-render
list.updateItem('item-1', {
    title: 'Updated Title',
    description: 'New description'
});

    Query and Retrieve

    // Get all items
const allItems = list.getItems();

// Get a specific item by ID
const item = list.getItemById('item-1');

// Find items matching criteria
const filtered = list.findItems(item =>
    item.title.includes('First')
);

    Custom Rendering

    // Define a custom render function
const renderItem = (item) => {
    const li = document.createElement('li');
    li.setAttribute('data-item-id', item.id);
    li.innerHTML = `
        <strong>${item.title}</strong>
        <em>${item.description}</em>
    `;
    return li;
};

// Add items with custom rendering
list.addItem({
    id: 'item-1',
    title: 'Custom Item',
    description: 'With custom HTML'
}, renderItem);

    Events

    The list helper dispatches custom events that you can listen to:

    const container = document.querySelector('.my-list');

// Listen for item addition
container.addEventListener('dynamic-item-added', (e) => {
    console.log('Item added:', e.detail.item);
});

// Listen for item removal
container.addEventListener('dynamic-item-removed', (e) => {
    console.log('Item removed:', e.detail.item);
});

// Listen for updates
container.addEventListener('dynamic-item-updated', (e) => {
    console.log('Item updated:', e.detail.item);
});

    DynamicPaginationHelper

    Dynamically manage pagination controls by adding and removing page links.

    Add Pages Dynamically

    import { DynamicPaginationHelper } from '../js/components/dynamic-content-helpers.js';

const pagination = new DynamicPaginationHelper('.pagination ul');

// Add pages at the end
pagination.addPageAtEnd({
    label: '2',
    href: '?page=2'
});

// Add pages at the beginning
pagination.addPageAtStart({
    label: '0',
    href: '?page=0'
});

    Remove Pages

    // Remove last page
pagination.removePageAtEnd();

// Remove first page
pagination.removePageAtStart();

// Remove specific page
pagination.removePageByNumber(3);

    Navigation

    // Set active page
pagination.setCurrentPage(2);

// Get total pages
const count = pagination.getPageCount();

// Get all pages
const pages = pagination.getPages();

    Events

    const paginationContainer = document.querySelector('.pagination ul');

// Listen for page changes
paginationContainer.addEventListener('pagination-page-changed', (e) => {
    console.log('Current page:', e.detail.page);
});

// Listen for page additions
paginationContainer.addEventListener('pagination-page-added', (e) => {
    console.log('Page added at:', e.detail.position);
});

    DynamicGridHelper

    Manage grid or card-based layouts for displaying dynamic content.

    Add Cards

    import { DynamicGridHelper } from '../js/components/dynamic-content-helpers.js';

const grid = new DynamicGridHelper('.card-grid');

// Add a card
grid.addCard({
    id: 'card-1',
    title: 'Card Title',
    description: 'Card description',
    icon: 'fas fa-star'
});

    Remove Cards

    // Remove by ID
grid.removeCard('card-1');

    Custom Card Rendering

    // Define custom card template
const renderCard = (card) => {
    const div = document.createElement('div');
    div.className = 'card custom';
    div.setAttribute('data-card-id', card.id);
    div.innerHTML = `
        <div class="card-header">
            <i class="${card.icon}"></i>
            <h3>${card.title}</h3>
        </div>
        <div class="card-body">
            <p>${card.description}</p>
        </div>
    `;
    return div;
};

// Add card with custom rendering
grid.addCard({
    id: 'card-1',
    title: 'Custom Card',
    description: 'Custom content',
    icon: 'fas fa-star'
}, renderCard);

    Complete Example: Todo List

    Here's a practical example of building a dynamic todo list:

    <html>
<head>
    <link rel="stylesheet" href="axiom.css">
</head>
<body>
    <div class="container">
        <h1>My Todo List</h1>
        <div style="margin: 20px 0;">
            <input type="text" id="todoInput" placeholder="Add a new todo">
            <button id="addBtn" class="button primary">Add</button>
        </div>
        <ul class="todo-list"></ul>
    </div>

    <script type="module">
        import { DynamicListHelper } from './js/components/dynamic-content-helpers.js';

        const todoList = new DynamicListHelper('.todo-list');
        const input = document.getElementById('todoInput');
        const addBtn = document.getElementById('addBtn');

        // Custom render function
        const renderTodo = (item) => {
            const li = document.createElement('li');
            li.setAttribute('data-item-id', item.id);
            li.className = 'todo-item';
            li.innerHTML = `
                <span>${item.title}</span>
                <button class="button small remove" type="button" data-remove-item="${item.id}">Remove</button>
            `;
            return li;
        };

        addBtn.addEventListener('click', () => {
            if (input.value.trim()) {
                todoList.addItem({
                    id: `todo-${Date.now()}`,
                    title: input.value
                }, renderTodo);
                input.value = '';
            }
        });

        document.querySelector('.todo-list').addEventListener('click', (event) => {
            const removeButton = event.target.closest('[data-remove-item]');
            if (!removeButton) return;
            todoList.removeItem(removeButton.getAttribute('data-remove-item'));
        });
    </script>
</body>
</html>

    Best Practices

    • Use unique IDs: Always provide unique IDs for items to avoid conflicts
    • Clean up events: Remove event listeners when components are destroyed
    • Custom rendering: Provide render functions for complex item structures
    • Error handling: Validate data before adding to prevent null/undefined errors
    • Performance: For large lists, consider batch operations using `addItems()`
    • Accessibility: Ensure rendered HTML maintains proper semantic structure
