Overview

Axicons is a native SVG icon library built specifically for Axiom01. With 200+ curated icons, zero external dependencies, and automatic theme support, Axicons is the perfect complement to the Axiom01 framework.

Native SVG

200+ SVG icons built directly into Axiom01. No external dependencies or CDN required.

Lightweight

Icons load instantly with zero network latency. Optimized for performance.

Theme-Aware

Icons automatically adapt to light and dark modes. Perfect theme integration.

Customizable

Use CSS variables and classes to style icons. Full control over appearance.

Accessible

Proper ARIA attributes and semantic HTML. Respects motion preferences.

Discoverable

Searchable icon browser with copy-to-clipboard. Find what you need fast.

Basic Usage

Simple Icon

The simplest way to use an icon is with two classes and a data attribute:

<span class="axicon render" data-name="Search"></span>

Result:

With Sizes

Use size utility classes to control icon size:

<span class="axicon render axicon-sm" data-name="Star"></span>
<span class="axicon render" data-name="Star"></span>
<span class="axicon render axicon-lg" data-name="Star"></span>
<span class="axicon render axicon-xl" data-name="Star"></span>

Result:

With Semantic Colors

Use semantic color classes to match Axiom's design tokens:

<span class="axicon render axicon-success" data-name="Check-Circle"></span>
<span class="axicon render axicon-warning" data-name="Alert-Triangle"></span>
<span class="axicon render axicon-error" data-name="X"></span>

Result:

In Buttons

Icons work perfectly in buttons and semantic elements:

<button>
    <span class="axicon render" data-name="Download"></span>
    Download
</button>

Result:

Customization

Size Utilities

Available size classes:

.axicon-sm   /* 0.875em */
.axicon      /* 1em (default) */
.axicon-lg   /* 1.25em */
.axicon-xl   /* 1.5em */
.axicon-2xl  /* 2em */

Color Utilities

Semantic color classes that use Axiom's design tokens:

.axicon-primary

.axicon-secondary

.axicon-success

.axicon-warning

.axicon-error

.axicon-info

Inline Styling

Override colors with inline styles for full flexibility:

<span class="axicon render" style="color: #e74c3c; width: 2rem; height: 2rem;" data-name="Heart"></span>

Result:

Animations

Use animation utilities for dynamic icons:

<!-- Spinning loader -->
<span class="axicon render axicon-spin axicon-lg" data-name="Loader"></span>

<!-- Pulsing notification -->
<span class="axicon render axicon-pulse axicon-lg" data-name="Bell"></span>

Result:

Dark Mode

Icons automatically adapt to dark mode via currentColor. No CSS changes needed:

<!-- Light mode: inherits dark text -->
<span class="axicon render" data-name="Search"></span>

<!-- Dark mode: automatically inherits light text -->
<!-- No changes required! -->
