"# Axiom01 Complete Component Library

## Overview
Axiom01 is a comprehensive, semantic-first UI framework featuring 70+ pre-built components organized into four core categories. All components follow consistent design patterns and integrate seamlessly with the framework's theming system.

---

## 1. INPUT CONTROLS

## GETTING STARTED

1. All components are accessed via `data-component` attributes
2. Components auto-initialize when the page loads
3. Access instances via `window.componentInstances.get(element)`
4. Listen to custom events for component interactions
5. Customize via CSS variables in `:root` or data attributes

### Example
```html
<div data-component="data-list" 
     data-source='[...]'
     data-view="grid">
</div>

<script>
  const el = document.querySelector('[data-component="data-list"]');
  const api = window.componentInstances.get(el);
  
  // Use API methods
  api.addItem({ title: 'New Item' });
</script>
```
