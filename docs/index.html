Axiom UI Framework

Craft beautiful interfaces with less markup. Minimal, semantic, flexible, and modern—Axiom is your new favorite UI toolkit.

Get Started

Getting Started

Install Axiom via CDN or npm, and start building with semantic HTML and minimal classes.

<link rel="stylesheet" href="https://your-cdn.com/axiom.min.css">
npm install axiom-ui

Include axiom.min.css in your project and start using Axiom classes.

Customization

Override variables, use utility classes, and extend Axiom to fit your brand. See axiom_vars.css for all available variables.

:root {
  --axiom-primary: #00c005;
  --axiom-radius: 6px;
  /* ... */
}
