Welcome to Axiom

The documentation hub for the Axiom UI framework.

Use the sidebar to navigate through the available guides and component documentation. This site is designed to be a comprehensive resource for learning and using Axiom to build beautiful, accessible, and performant web interfaces.

Our Philosophy

Axiom is built on a foundation of semantic HTML and a minimal class approach. We believe that a framework should empower you, not get in your way. By leveraging the power of CSS custom properties, we provide a highly themeable and flexible system that encourages best practices and results in cleaner, more maintainable code.

Getting Started

To get started, we recommend exploring the following sections:

  • Theming & Colors: Learn how to customize the look and feel of Axiom.
  • Components: Browse the extensive library of ready-to-use UI components.
  • Advanced Guides: Dive deeper into topics like forms, media, and typography.
