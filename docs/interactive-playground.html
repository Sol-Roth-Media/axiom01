Interactive Playground

Welcome to the Axiom01 Interactive Playground! This tool allows you to write HTML and CSS directly in your browser and see the results rendered live. It's a great way to experiment with Axiom01 components, test custom styles, and quickly prototype ideas.

How to Use

  1. Edit the HTML and CSS in the left editor.
  2. Click "Run Code" to update the live preview on the right.
  3. The preview automatically includes Axiom01's core CSS.
  4. You can also select different themes to see how your code adapts.
