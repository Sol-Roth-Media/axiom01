Axicons

Base + premium icon sets for Axiom01. Click any icon to copy its full SVG.

Getting Started

Load only what you need. Start with the base set and add variants as required.

HTML Markup

<!-- Basic icon -->
<span class="axicon render" data-name="Check"></span>

<!-- Premium icon example -->
<span class="axicon render" data-name="Check-Premium"></span>

Script Loading

<!-- Base icon set - always load -->
<script src="js/axicons-base.js"></script>

<!-- Optional premium extension -->
<script src="js/axicons-premium.js"></script>

<!-- Renderer (always last) -->
<script src="js/render-icons.js"></script>

Styling

/* Icons inherit currentColor */
.axicon svg {
  width: 1em;
  height: 1em;
}

/* Color via parent */
button {
  color: var(--a-color-primary);
}

/* Explicit size */
.icon-lg .axicon { font-size: 2rem; }

Icon Color & Theme Adaptation

Base icons use currentColor for all strokes and fills — they automatically inherit the text color of their parent and adapt to light/dark themes with zero extra CSS.

How It Works

Every base icon is drawn with stroke="currentColor". Setting color on a parent element is all you need — the icon follows automatically, including when Axiom switches between light and dark themes.

/* Icon inherits button's text color */
.my-button {
  color: var(--a-color-primary);
}

/* Dark theme: icons go light automatically */
[data-theme="dark"] .axicon { /* no extra work needed */ }

Semantic Color Classes

Apply Axiom color modifier classes directly on the .axicon to use a design-token color.

<!-- Primary brand color -->
<span class="axicon render primary" data-name="Check"></span>

<!-- Success green -->
<span class="axicon render success" data-name="Check"></span>

<!-- Error red -->
<span class="axicon render error" data-name="Alert"></span>

Premium Icons

Premium icons use hardcoded gradients for rich visual moments — they intentionally do not respond to currentColor. Use them as accent or hero elements, not inline with body text.

<!-- Always renders its gradient -->
<span class="axicon render" data-name="AX10M-Premium"></span>

<!-- Adapts to theme automatically -->
<span class="axicon render" data-name="Axiom-System"></span>
