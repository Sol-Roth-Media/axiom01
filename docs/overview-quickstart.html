Overview & Quickstart

Get started with Axiom01 in minutes. A semantic-first UI framework that respects your code.

What is Axiom01?

Axiom01 is a semantic-first UI framework that prioritizes accessibility, flexibility, and clean code. Write less code and build better interfaces without utility chaos or BEM complexity.

73 Components

Button, Card, Modal, Table, Video Player, AI Chat, and more

149 Tokens

Colors, Spacing, Typography, Shadows, Animations

13.3 KB CSS

Everything included. Minimal, performant, powerful.

Why Axiom01 is Different

Semantic-First

Write meaningful HTML using <article>, <section>, <header>. Axiom understands your content.

Minimal Classes

One class per component. class="card" instead of class="max-w-sm rounded shadow-lg bg-white p-6".

Accessible by Default

WCAG 2.1 AA out of the box. Keyboard navigation, focus states, color contrast all handled.

Installation

Get Axiom01 up and running in your project:

Via npm

npm install axiom01

Via CDN

Add these links to your HTML <head>:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/axiom01/dist/css/axiom.min.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/axiom01/dist/css/axicons.min.css">
<script src="https://cdn.jsdelivr.net/npm/axiom01/dist/js/axicons.min.js"></script>

Basic Button

Once installed, you can create components with simple HTML:

<button class="primary">Click Me</button>

Quick Example: Card Component

Here's how semantic Axiom01 HTML compares to utility-first approaches:

Axiom01 Approach

HTML: 8 lines | Classes: 1

<article class="card">
  <header>
    <h3>Title</h3>
  </header>
  <main>
    <p>Content here.</p>
  </main>
  <footer>
    <button class="primary">Action</button>
  </footer>
</article>

Utility-First Approach

HTML: 10 lines | Classes: 18

<div class="max-w-sm rounded shadow bg-white p-6">
  <div class="border-b pb-4 mb-4">
    <h3 class="text-lg font-bold">Title</h3>
  </div>
  <div class="mb-4">
    <p class="text-gray-600">Content here.</p>
  </div>
  <div class="flex justify-end">
    <button class="px-4 py-2 bg-blue-500">
      Action
    </button>
  </div>
</div>
