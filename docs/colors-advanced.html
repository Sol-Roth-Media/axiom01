Color System

Axiom01's color system is mathematically designed to ensure harmony, accessibility, and flexibility across your entire interface.

Color Philosophy

Axiom01 uses a sophisticated color system based on semantic colors and mathematical relationships. Every color in the system works with every other color, ensuring that your interfaces are always harmonious and accessible.

The color system supports:

  • Semantic Colors - Primary, Secondary, Success, Warning, Error, and Info colors that communicate intent
  • Automatic Derivations - Light variants, containers, and text colors are computed from base colors
  • Light & Dark Modes - Colors adapt automatically with proper contrast ratios
  • Accessible Contrast - WCAG 2.1 AA compliance built-in
  • Multiple Palettes - Switch between default, indigo, emerald, and sunset palettes

The primary color is the most important color in your design system. It's used for interactive elements, links, and key actions. Axiom01 provides multiple primary color options.

Primary Color Variations

For each primary color, Axiom01 automatically creates these variations:

--a-color-primary: #2563eb;              /* Base primary color */
--a-color-on-primary: #ffffff;           /* Text color ON primary */
--a-color-primary-light: lighter shade;  /* Light variant for backgrounds */
--a-color-primary-dark: darker shade;    /* Dark variant for emphasis */
--a-color-primary-container: very light; /* Container background */
--a-color-on-primary-container: dark;    /* Text ON container */

Text Hierarchy

--a-color-text-primary: Main body text (100% contrast)
--a-color-text-secondary: Secondary content (70% opacity)
--a-color-text-muted: Supplementary info (50% opacity)
--a-color-text-disabled: Disabled state (40% opacity)

Switching Color Palettes

Change the primary color palette dynamically by setting the data-axiom-palette attribute on the root element.

<!-- Default Blue -->
<html data-axiom-palette="default">

<!-- Indigo -->
<html data-axiom-palette="indigo">

<!-- Emerald -->
<html data-axiom-palette="emerald">

<!-- Sunset -->
<html data-axiom-palette="sunset">

JavaScript Palette Switching

// Change palette at runtime
document.documentElement.setAttribute('data-axiom-palette', 'emerald');

// Or toggle dark mode
document.documentElement.setAttribute('data-theme', 'dark');

Accessibility & Contrast

All colors in Axiom01 are chosen to meet WCAG 2.1 AA accessibility standards. This ensures:

  • Minimum 4.5:1 contrast ratio for normal text
  • Minimum 3:1 contrast ratio for large text
  • Automatic dark mode adaptation maintains contrast
  • Color-blind friendly semantic meanings (not just color-based)

Always pair colors with meaningful text or icons to communicate intent, not just color alone.
