Color System

Mathematically designed color system with accessibility and harmony built-in

Color Philosophy

Axiom01's color system is mathematically designed to ensure harmony, accessibility, and flexibility across your entire interface. Every color works with every other color, ensuring your interfaces are always harmonious and accessible.

Key Features

  • Semantic Colors — Primary, Secondary, Success, Warning, Error, and Info colors that communicate intent
  • Automatic Derivations — Light variants, containers, and text colors computed from base colors
  • Light & Dark Modes — Colors adapt automatically with proper contrast ratios
  • Accessible Contrast — WCAG 2.1 AA compliance built-in
  • Multiple Palettes — Switch between default, indigo, emerald, and sunset palettes

Primary Color Variations

For each primary color, Axiom01 automatically creates these CSS variables:

--a-color-primary: #2563eb;              /* Base primary color */
--a-color-on-primary: #ffffff;           /* Text color ON primary */
--a-color-primary-light: lighter shade;  /* Light variant for backgrounds */
--a-color-primary-dark: darker shade;    /* Dark variant for emphasis */
--a-color-primary-container: very light; /* Container background */
--a-color-on-primary-container: dark;    /* Text ON container */

Surface & Text Colors

Surface and text colors define the foundation of your interface. They adapt automatically for light and dark modes while maintaining accessibility standards.

Light Mode

Surface

--a-color-surface

Text (Primary)

--a-color-text

Text Hierarchy

--a-color-text: Main body text (100% contrast)
--a-color-on-surface-variant: Secondary content (70% opacity)
--a-color-outline: Borders and dividers (50% opacity)
--a-color-on-surface: Tertiary text

Switching Color Palettes

Change the primary color palette dynamically by setting the data-axiom-palette attribute on the root element.

<!-- Default Blue -->
<html data-axiom-palette="default">

<!-- Indigo -->
<html data-axiom-palette="indigo">

<!-- Emerald -->
<html data-axiom-palette="emerald">

<!-- Sunset -->
<html data-axiom-palette="sunset">

JavaScript Palette Switching

// Change palette at runtime
document.documentElement.setAttribute('data-axiom-palette', 'emerald');

// Or toggle dark mode
document.documentElement.setAttribute('data-theme', 'dark');

Accessibility & Contrast

All colors in Axiom01 meet WCAG 2.1 AA accessibility standards:

  • Minimum 4.5:1 contrast ratio for normal text
  • Minimum 3:1 contrast ratio for large text and UI components
  • Automatic dark mode adaptation maintains contrast in both themes
  • Color-blind friendly semantic meanings, not color alone

Best Practice: Always pair colors with meaningful text or icons to communicate intent, not just color alone.
