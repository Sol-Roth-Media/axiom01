Axiom UI Components

Explore the components available in the Axiom UI Framework. Each component follows Axiom's philosophy of semantic HTML, minimal classes, and CSS variables for customization.

Buttons

Buttons are used to trigger actions or navigate to different pages. Axiom provides several button variants with minimal markup.

<button class="button">Default Button</button>
<button class="button primary">Primary Button</button>
<button class="button secondary">Secondary Button</button>
<button class="button success">Success Button</button>
<button class="button danger">Danger Button</button>
<button class="button" disabled>Disabled Button</button>
Class Description
.button Base button class
.primary Primary action button
.secondary Secondary action button
.success Success action button
.danger Dangerous action button

Cards

Cards are used to group related content and actions. Axiom uses semantic HTML elements inside a card container.

Card Title
This is a minimal card using Axiom's semantic approach. Cards use article, header, section, and footer elements.
<article class="card">
  <header>Card Title</header>
  <section>This is a minimal card using Axiom's semantic approach. Cards use article, header, section, and footer elements.</section>
  <footer>
    <button class="button small">Action</button>
  </footer>
</article>
Element/Class Description
.card Base card class
<header> Card header (title area)
<section> Card content area
<footer> Card footer (action area)

Alerts

Alerts are used to display important messages to the user. Axiom provides several alert variants for different types of messages.

This is a default alert message.
This is a success alert message.
This is a warning alert message.
This is a danger alert message.
This is an info alert message.
<div class="alert">This is a default alert message.</div>
<div class="alert success">This is a success alert message.</div>
<div class="alert warning">This is a warning alert message.</div>
<div class="alert danger">This is a danger alert message.</div>
<div class="alert info">This is an info alert message.</div>
Class Description
.alert Base alert class
.success Success message alert
.warning Warning message alert
.danger Error/danger message alert
.info Information message alert
