Form Elements

Axiom01 provides a clean, accessible, and highly customizable base for common form elements. We leverage native HTML elements as much as possible, applying minimal styling to ensure consistency across browsers while allowing for easy customization via CSS variables.

Text Inputs

Standard text input fields for various types of user data.

This is some helpful text for the input.

This field has an error.

<div class="form-group">
  <label for="textInput">Text Input</label>
  <input type="text" id="textInput" placeholder="Enter text here">
  <p class="help-text">This is some helpful text for the input.</p>
</div>
<div class="form-group">
  <label for="disabledInput">Disabled Input</label>
  <input type="text" id="disabledInput" value="Cannot edit this" disabled>
</div>
<div class="form-group">
  <label for="errorInput">Input with Error</label>
  <input type="text" id="errorInput" value="Invalid data" class="error" aria-invalid="true" aria-describedby="errorInputHelp">
  <p id="errorInputHelp" class="error-text">This field has an error.</p>
</div>

Textarea

Multi-line text input for longer content.

<div class="form-group">
  <label for="textareaInput">Textarea</label>
  <textarea id="textareaInput" rows="5" placeholder="Enter your message here..."></textarea>
</div>
<div class="form-group">
  <label for="disabledTextarea">Disabled Textarea</label>
  <textarea id="disabledTextarea" rows="3" disabled>This textarea is disabled.</textarea>
</div>

Select Dropdown

Standard dropdown for selecting options.

<div class="form-group">
  <label for="selectInput">Select Option</label>
  <select id="selectInput">
    <option value="">--Please choose an option--</option>
    <option value="option1">Option 1</option>
    <option value="option2">Option 2</option>
    <option value="option3">Option 3</option>
  </select>
</div>
<div class="form-group">
  <label for="disabledSelect">Disabled Select</label>
  <select id="disabledSelect" disabled>
    <option value="">--Cannot select--</option>
  </select>
</div>
