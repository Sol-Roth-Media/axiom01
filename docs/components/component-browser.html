Component Browser

Preview a curated set of Axiom01 components in one place, then filter and open individual demos when you want the full documentation page.

Live Example

Preview Components

4 demos available in this sample browser.

Markup Structure

<div class="component-browser">
    <header>
        <div class="info">
            <h4>Preview Components</h4>
            <p><span id="component-count">4</span> demos available.</p>
        </div>
        <button id="view-fullscreen-button" type="button">Open Demo</button>
    </header>
    <div class="controls">
        <input id="component-search-input" type="search" />
        <select id="component-category-filter">...</select>
    </div>
    <div class="tabs-container">
        <div class="tabs" role="tablist">
            <button class="tab-button active" type="button" data-src="button.html">Button</button>
        </div>
    </div>
    <div class="preview-container">
        <div class="preview">
            <iframe id="component-preview-iframe" title="Component preview"></iframe>
        </div>
    </div>
</div>

Why Use It

  • Compare multiple component demos without leaving the page.
  • Filter a preview set by category before opening the full docs page.
  • Reuse the same ids and class hooks that the shared docs runtime already supports.
