Component Browser

The Component Browser is a meta-component used to showcase and interact with other Axiom01 components. It features tabbed navigation, search, and category filtering, providing a dynamic way to explore the framework's library.

Usage Guidelines

The Component Browser is typically used for documentation sites. The data-component="component-browser" attribute triggers its JavaScript functionality, which handles tab switching, iframe loading, and search/filter logic.

Each component tab is a .tab-button with a data-src attribute pointing to the HTML file of the component to be previewed. The data-category attribute is used for filtering.

The actual component preview is rendered inside an <iframe> with the ID component-preview-iframe.
