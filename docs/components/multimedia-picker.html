Basic Usage

Select or drag files, then preview image, video, and audio assets.

    Required Markup Contract

    • Use data-component="multimediapicker" on the root container.
    • Include one file input and one preview list with data-role="list".
    • Include a status element with role="status" for upload/selection feedback.
