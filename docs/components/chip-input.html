Chip Input

Multi-value input component for tags, labels, and multi-select fields. Perfect for email recipients, content tags, and dynamic form inputs with visual feedback.

Chip Input Component

The Chip Input component allows users to enter and manage multiple values as individual "chips" or "tags". Each value appears as a removable badge, making it ideal for email fields, tag inputs, and multi-select scenarios.

Basic Example

React Vue 
<div class="chip-input">
  <span class="chip">React <button aria-label="Remove React">×</button></span>
  <span class="chip">Vue <button aria-label="Remove Vue">×</button></span>
  <input type="text" class="chip-input-field" placeholder="Add more tags...">
</div>

Form Integration

Add Recipients
alice@example.com bob@example.com 
<div class="chip-input" data-chip-input="recipients">
  <span class="chip">alice@example.com <button>×</button></span>
  <input type="email" class="chip-input-field" placeholder="Enter email...">
</div>
<input type="hidden" name="recipients">

Use Cases

  • Email recipient fields
  • Content tagging systems
  • Multi-select form inputs
  • Skill or interest selection
  • Filter tag management
  • Category selection in CMSs

JavaScript Implementation

The Chip Input component requires JavaScript to handle adding/removing chips and updating the associated hidden form field. Include js/axiom.js for automatic initialization.

Configuration Options

Attribute Value Description
data-chip-input field name Links to hidden input field for form sync
data-separator character (default: comma) Delimiter for parsing input values
aria-label descriptive text Screen reader accessible label

Browser Support

All modern browsers. Requires ES6+ JavaScript for full functionality.
