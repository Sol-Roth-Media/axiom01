Chip Input

Axiom01's chip input component enables users to enter and manage multiple values (tags, keywords, recipients). Features auto-complete suggestions, removal, and visual feedback.

HTML Structure

<div class="chip-input-wrapper">
    <!-- Existing chips -->
    <div class="chip">
        Tag Name
        <button class="chip-remove" aria-label="Remove">&times;</button>
    </div>
    
    <!-- Input field -->
    <input type="text" class="chip-input" 
           placeholder="Add more...">
</div>

<!-- Suggestions (optional) -->
<div class="chip-suggestions">
    <ul class="suggestion-list">
        <li class="suggestion-chip">Suggestion 1</li>
        <li class="suggestion-chip">Suggestion 2</li>
    </ul>
</div>

Features

  • Multiple Chips - Add and manage multiple values
  • Remove Buttons - Easy deletion with X button
  • Auto-complete - Suggestions for quick input
  • Form Integration - Works as form input
  • Visual Feedback - Clear chip styling
  • Flexible Content - Text, emails, tags, etc.

Use Cases

  • Tags/Keywords - Article and blog tagging
  • Email Recipients - To, CC, BCC fields
  • User Selection - Invite team members
  • Skills/Interests - User profile setup
  • Permissions - Grant access to users
  • Filtering - Multi-select filter tags

Accessibility

  • Use semantic wrapper for input container
  • Include aria-label on remove buttons
  • Support keyboard (Space/Enter to add, Backspace to remove)
  • Announce chip additions/removals to screen readers
  • Maintain focus management when removing chips
  • Provide descriptive labels for the input

Best Practices

  • Clear Removal - Always provide remove buttons
  • Input Validation - Prevent duplicate entries
  • Suggestions - Show relevant auto-complete options
  • Visual Distinction - Make chips clearly visible
  • Keyboard Support - Full keyboard accessibility
  • Count Display - Show number of chips added
