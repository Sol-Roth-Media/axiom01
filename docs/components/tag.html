Tag Component

Label components for categorization, filtering, and content organization with interactive and removable variants.

Basic Tags

Simple labels for content categorization.

JavaScript CSS HTML Accessibility 
<div class="tags" role="group" aria-label="Content tags">
  <span class="tag">JavaScript</span>
  <span class="tag">CSS</span>
  <span class="tag">HTML</span>
  <span class="tag">Accessibility</span>
</div>

Interactive Tags

Clickable tags for filtering and selection.

<div class="tags" role="group" aria-label="Filter tags">
  <button class="tag" aria-pressed="false">
    <span>Frontend</span>
  </button>
  <button class="tag active" aria-pressed="true">
    <span>Backend</span>
  </button>
</div>

Removable Tags

Tags with remove functionality for dynamic content management.

React Node.js TypeScript 
<span class="tag removable">
  <span>React</span>
  <button class="remove" aria-label="Remove React tag">
    <i class="fas fa-times" aria-hidden="true"></i>
  </button>
</span>

Tag Input

Input field for adding new tags dynamically.

JavaScript Python 
<div class="tag-input">
  <label for="skill-tags">Add Skills</label>
  <div class="tags editable">
    <span class="tag removable">
      <span>JavaScript</span>
      <button class="remove" aria-label="Remove JavaScript tag">
        <i class="fas fa-times" aria-hidden="true"></i>
      </button>
    </span>
    <input type="text" id="skill-tags" placeholder="Add a skill..." aria-label="Add new skill tag">
  </div>
</div>

Accessibility Features

  • Keyboard Navigation: Interactive tags support Enter and Space key activation
  • ARIA States: Proper pressed states for toggle functionality
  • Screen Reader Support: Clear labels for remove buttons and tag groups
  • Focus Management: Logical tab order through tag collections
  • Semantic Structure: Proper use of buttons vs spans based on interactivity
