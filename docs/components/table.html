Table

Tables are used to organize and display data in a structured, grid-like format.

Basic Table

The default table style is clean and simple, with clear headings and row dividers.

# First Name Last Name Handle
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter 
<table>
    <thead>
        <tr>
            <th>#</th>
            <th>First Name</th>
            ...
        </tr>
    </thead>
    <tbody>
        <tr>
            <th>1</th>
            <td>Mark</td>
            ...
        </tr>
    </tbody>
</table>

Striped Rows

Add the `.striped` class to the `

` element for alternating row colors, which can improve readability for large data sets.

# First Name Last Name Handle
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter 
<table class="striped">
    ...
</table>
