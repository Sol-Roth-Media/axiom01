Carousel Component

Axiom01's carousel component is a flexible, responsive slideshow for cycling through a series of images or content. It's designed to be accessible and easily integrated into your web projects.

Usage Guidelines

The carousel component requires JavaScript for its interactive functionality (sliding, controls, indicators). Ensure axiom.js is loaded. The data-component="carousel" attribute will initialize the component.

Each .carousel-item represents a slide. The .active class determines the initially visible slide. Navigation is handled by .carousel-control-prev and .carousel-control-next, while .carousel-indicators provide direct access to slides.
