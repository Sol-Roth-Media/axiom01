HTML Structure

Basic Carousel

<div role="region" aria-label="Carousel" class="carousel">
  <div class="carousel-viewport">
    <div class="carousel-track">
      <figure>
        <img src="image-1.jpg" alt="Slide 1">
        <figcaption>Slide 1</figcaption>
      </figure>
      <figure>
        <img src="image-2.jpg" alt="Slide 2">
        <figcaption>Slide 2</figcaption>
      </figure>
    </div>
  </div>
  <button class="carousel-control prev">Previous</button>
  <button class="carousel-control next">Next</button>
  <div class="carousel-indicators">
    <button data-active="true"></button>
    <button></button>
  </div>
</div>

</div>

CSS Classes & Data Attributes

.carousel

Main container with role="region" for accessibility.

.carousel-viewport

Visible container, clips overflow.

.carousel-track

Flex container holding slides.

.carousel-control

Previous/next navigation buttons.

.carousel-indicators

Dot indicators at bottom.

data-autoplay="true"

Auto-advance slides every 5 seconds.

data-items="n"

Show n items at once instead of 1.

data-variant="vertical"

Vertical scroll instead of horizontal.

JavaScript API

Navigate to Slide

const carousel = document.querySelector('.carousel');
const track = carousel.querySelector('.carousel-track');

function goToSlide(index) {
  const offset = index * 100;
  track.style.transform = `translateX(-${offset}%)`;
  
  // Update indicators
  const indicators = carousel.querySelectorAll('.carousel-indicators button');
  indicators.forEach((btn, i) => {
    if (i === index) {
      btn.setAttribute('data-active', 'true');
    } else {
      btn.removeAttribute('data-active');
    }
  });
}

Next/Previous Controls

let currentIndex = 0;
const slides = carousel.querySelectorAll('.carousel-track figure');

function nextSlide() {
  currentIndex = (currentIndex + 1) % slides.length;
  goToSlide(currentIndex);
}

function prevSlide() {
  currentIndex = (currentIndex - 1 + slides.length) % slides.length;
  goToSlide(currentIndex);
}

carousel.querySelector('.carousel-control.next').addEventListener('click', nextSlide);
carousel.querySelector('.carousel-control.prev').addEventListener('click', prevSlide);

Accessibility

Best Practices

  • Use role="region" with descriptive aria-label
  • Provide text alternatives for images (alt text)
  • Label navigation buttons with aria-label
  • Use <figure> and <figcaption> semantically
  • Pause auto-play on user interaction
  • Keyboard navigation (arrow keys)
  • Announce slide changes to screen readers
  • Provide skip controls for auto-play

AXIOM01 Philosophy

Why Use Semantic Carousels

Carousels embody AXIOM01 principles:

  • Semantic: <figure> and <figcaption> indicate gallery items
  • Accessible: role="region" helps screen readers understand content
  • Progressive: Works without JavaScript, enhanced with JS
  • Data-Driven: Configuration via data attributes
  • Flexible: Adapts to single/multiple items, horizontal/vertical
  • Responsive: CSS handles layout, JS handles interaction

Avoid custom carousel solutions that ignore semantics. Use proper HTML structure.
