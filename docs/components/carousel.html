Carousel

Axiom01's carousel component displays content in a rotating slideshow format. Supports auto-play, manual navigation, and indicator dots. Perfect for featured content, testimonials, and image galleries.

    <div class="carousel-inner">
        <div class="carousel-item active">
            <div class="carousel-icon">🚀</div>
            <h3 class="carousel-title">Title</h3>
            <p class="carousel-description">Description</p>
        </div>
        <div class="carousel-item">
            <!-- More items -->
        </div>
    </div>
    
    <div class="carousel-controls">
        <button class="carousel-button" id="prevBtn">Previous</button>

Features

  • Auto-Play - Optional automatic slide progression
  • Manual Navigation - Previous/Next buttons
  • Indicator Dots - Visual slide position tracking
  • Keyboard Navigation - Arrow keys to navigate
  • Fade Transitions - Smooth slide animations
  • Counter Display - Show current slide number

Use Cases

  • Featured Content - Showcase hero content
  • Testimonials - Rotate customer quotes
  • Image Gallery - Display photo collections
  • Product Showcase - Highlight products
  • Onboarding - Step-through tutorial slides
  • News Headlines - Rotating news ticker

Accessibility

  • Support keyboard navigation (arrow keys)
  • Include proper ARIA labels on buttons
  • Announce slide changes to screen readers
  • Pause auto-play on keyboard interaction
  • Visible focus indicators on all controls
  • Provide skip option for auto-playing carousels

Best Practices

  • Clear Navigation - Easy prev/next buttons
  • Visual Indicators - Show current slide position
  • Auto-Play Caution - Don't use for critical content
  • Smooth Transitions - Not too fast or slow
  • Touch Support - Swipe gestures on mobile
  • Content Sizing - Consistent item heights
