HTML Structure

<!-- Top Navigation -->
<nav class="navbar" role="navigation">
    <div class="navbar-brand">Brand</div>
    <ul class="navbar-menu">
        <li class="navbar-menu-item">
            <a href="#" class="navbar-link active">Home</a>
        </li>
    </ul>
</nav>

<!-- Breadcrumb -->
<div class="breadcrumb-nav">
    <ul class="breadcrumb-list">
        <li><a href="#" class="breadcrumb-link">Home</a></li>
        <li><span class="breadcrumb-separator">/</span></li>
        <li><span>Current</span></li>
    </ul>
</div>

<!-- Tabs -->
<div class="tab-nav" role="tablist">
    <button class="tab-link active" role="tab">Tab 1</button>
    <button class="tab-link" role="tab">Tab 2</button>
</div>

Navigation Types

  • Top Navigation - Horizontal menu bar with logo and links
  • Breadcrumb - Shows current location in hierarchy
  • Tab Navigation - Switch between related content sections
  • Sidebar Navigation - Vertical menu on left side
  • Footer Navigation - Links grouped in footer
  • Mobile Menu - Hamburger menu for responsive

Use Cases

  • Website Navigation - Main site menu structure
  • Application Navigation - App section switching
  • Documentation - Chapter/section navigation
  • E-commerce - Product category browsing
  • Content Management - Page hierarchy navigation
  • Nested Navigation - Multiple levels of menus

Accessibility

  • Use semantic <nav> element with role and aria-label
  • Include role="tablist" for tab navigation
  • Use aria-selected on active tab
  • Support keyboard navigation (Tab, Arrow keys)
  • Highlight current page/section visually
  • Provide skip navigation links

Best Practices

  • Clear Hierarchy - Organize logically with clear labels
  • Active Indicators - Show current location/page
  • Responsive Design - Stack or hide on mobile
  • Icon + Text - Combine for better clarity
  • Keyboard Navigation - Full keyboard support required
  • Consistent Placement - Keep navigation in same location
