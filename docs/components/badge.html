Badge

Axiom01's badge component displays labels, tags, and status indicators. Perfect for categorization, counts, and visual labels.

Live Examples

New Active Pending Error Label 99+ Featured

Features

  • Multiple Types - Primary, success, warning, error, outline
  • Compact - Small, inline display
  • Icon Support - Optional icons before/after
  • Flexible Text - Numbers, words, or emojis
  • Color Coded - Visual type indication
  • Semantic - Convey meaning through appearance

Use Cases

  • Status Indicators - Active, pending, error
  • Tags - Category labels
  • Counts - Notification or item counts
  • Badges - Achievement or feature badges
  • Labels - Generic text labels
  • Keywords - Search or filter tags

Accessibility

  • Use clear, descriptive text
  • Color + text for meaning
  • Sufficient contrast (WCAG AA)
  • Don't use color alone for meaning
  • Include in semantic structure
  • Announce counts if important

Best Practices

  • Clear Label - Text should be concise
  • Consistent Sizing - Uniform badge dimensions
  • Color Semantics - Match colors to meaning
  • Positioned Well - Place near relevant content
  • Not Actionable - Badges shouldn't be buttons
  • Performance - Keep text minimal
