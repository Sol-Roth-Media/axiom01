Badge Component

Badges are small count and labeling components. They are used to highlight information like the number of notifications or the status of an item.

Standard Badges

Use the `.badge` class along with contextual color classes to create standard badges.

Default Primary Secondary Success Warning Error 
<span class="badge">Default</span>
<span class="badge primary">Primary</span>
<span class="badge secondary">Secondary</span>
<span class="badge success">Success</span>
<span class="badge warning">Warning</span>
<span class="badge error">Error</span>

Pill Badges

Use the `.badge.pill` classes for a more rounded badge style.

Default Primary Secondary Success Warning Error 
<span class="badge pill">Default</span>
<span class="badge pill primary">Primary</span>

Badges with Icons

Badges can include icons for additional visual context.

Success Error 
<span class="badge success"><i class="fas fa-check-circle"></i> Success</span>
<span class="badge error"><i class="fas fa-times-circle"></i> Error</span>

Usage with Other Components

Badges are often used with other components, like buttons and navigation links.

Inbox 99+ 
<button class="button">
    Notifications <span class="badge primary">4</span>
</button>

<a href="#" class="button secondary">
    Inbox <span class="badge pill">99+</span>
</a>
