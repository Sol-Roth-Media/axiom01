Toggle

Axiom01's toggle component provides a switch for binary on/off states. Perfect for settings, preferences, and boolean options.

HTML Structure

<div class="toggle-row">
    <span class="toggle-label">Feature Name</span>
    <div class="switch">
        <div class="switch-thumb"></div>
    </div>
    <span class="status-indicator">Off</span>
</div>

Features

  • Binary State - On/off switch control
  • Smooth Animation - Fluid thumb movement
  • Status Indicator - Show current state
  • Keyboard Support - Space to toggle
  • Accessible - Full ARIA support
  • Hover Effects - Visual feedback

Use Cases

  • Settings - Feature toggles
  • Preferences - User options
  • Notifications - Enable/disable alerts
  • Theme Switcher - Dark/light mode
  • Permissions - Access control
  • Features - Beta or experimental features

Accessibility

  • Use role="switch" and aria-checked
  • Support keyboard toggle with Space/Enter
  • Include descriptive labels
  • Announce state to screen readers
  • Visible focus indicator
  • Color + text for state indication

Best Practices

  • Clear Labels - Describe what toggles
  • Status Display - Show On/Off state
  • Immediate Feedback - Respond instantly
  • Consistent Style - Match component set
  • Proper Sizing - Touch-friendly (44×44px)
  • Save State - Persist user preference
