Chat Component

Axiom01's chat component provides a clean and functional interface for displaying conversational messages. It's designed to be easily integrated into various applications, from customer support widgets to internal communication tools.

Basic Chat Widget

Hello! How can I help you today?
I need assistance with my order.
Certainly! Could you please provide your order number?
It's #AXM012345.
Thank you. Please wait a moment while I retrieve the details.

Code Example

<div class="chat-widget" data-component="chat">
    <div class="chat-history">
        <div class="chat-message bot">Hello! How can I help you today?</div>
        <div class="chat-message user">I need assistance with my order.</div>
        <div class="chat-message bot">Certainly! Could you please provide your order number?</div>
        <div class="chat-message user">It's #AXM012345.</div>
        <div class="chat-message bot">Thank you. Please wait a moment while I retrieve the details.</div>
    </div>
    <form class="chat-form">
        <input type="text" class="chat-input" placeholder="Type your message...">
        <button class="button primary">Send</button>
    </form>
</div>

Usage Guidelines

The .chat-widget class defines the main container. Message history is within .chat-history, with individual messages using .chat-message. Differentiate user and bot messages with .user and .bot modifier classes respectively. The input area is structured with .chat-form, .chat-input, and a standard .button.

You will need JavaScript to handle sending and receiving messages, scrolling to the latest message, and dynamically adding new messages to the .chat-history.
