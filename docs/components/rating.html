Features

  • Star Rating - 1-5 star scale
  • Interactive - Click to set rating
  • Read-Only - Display existing ratings
  • Hover Preview - Show rating before click
  • Statistics - Distribution visualization
  • Average Display - Overall rating summary

Use Cases

  • Product Reviews - E-commerce ratings
  • User Feedback - Service quality rating
  • Satisfaction Survey - Experience rating
  • Content Rating - Article or video rating
  • App Rating - App store ratings
  • Support Quality - Service feedback

Accessibility

  • Use role="radiogroup" for rating group
  • Include aria-label on each star
  • Support keyboard navigation (arrow keys)
  • Announce selected rating to screen readers
  • Provide text alternative to star icons
  • Support focus management

Best Practices

  • Clear Scale - Show that it's 1-5 stars
  • Hover State - Preview rating before click
  • Confirmation - Show current rating after submission
  • Distribution - Show rating breakdown
  • Count Display - Show number of ratings
  • Disabled State - Show when rating unavailable
