Dropdown Component

Axiom01's dropdown component provides a toggleable, contextual overlay for displaying lists of links or actions. It's designed to be accessible and easy to integrate into navigation menus or action buttons.

Basic Dropdown

  • Action
  • Another action
  • Something else here
  • Separated link

Code Example

<div class="dropdown" data-component="dropdown">
    <button class="button primary dropdown-toggle" aria-expanded="false" aria-haspopup="true">
        <span>Dropdown Menu</span>
        <i class="fas fa-chevron-down"></i>
    </button>
    <ul class="dropdown-menu" role="menu">
        <li><a href="#" role="menuitem">Action</a></li>
        <li><a href="#" role="menuitem">Another action</a></li>
        <li><a href="#" role="menuitem">Something else here</a></li>
        <li class="divider" role="separator"></li>
        <li><a href="#" role="menuitem">Separated link</a></li>
    </ul>
</div>

Usage Guidelines

The .dropdown class wraps the entire component. The .dropdown-toggle button triggers the dropdown menu. The actual menu is an unordered list with the class .dropdown-menu.

The data-component="dropdown" attribute will initialize the necessary JavaScript to toggle the is-open class on the dropdown, manage ARIA attributes, and handle outside clicks for dismissal.

Accessibility is handled with aria-expanded and aria-haspopup attributes on the toggle button, and role="menu" and role="menuitem" on the menu and its items respectively.
