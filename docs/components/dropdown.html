Dropdown

Dropdowns are toggleable overlays that allow the user to choose from a list of options, often used for navigation or action menus.

Basic Dropdown

A standard dropdown consists of a toggle button and a menu of links. The component is initialized with `data-component="dropdown"`.

  • Action
  • Another action
  • Something else here
  • Separated link
<div class="dropdown" data-component="dropdown">
    <button class="dropdown-toggle primary">
        <span>Dropdown Menu</span>
        <i class="fas fa-chevron-down"></i>
    </button>
    <ul class="dropdown-menu">
        <li><a href="#">Action</a></li>
        <li><a href="#">Another action</a></li>
        <li class="divider"></li>
        <li><a href="#">Separated link</a></li>
    </ul>
</div>

Best Practices

Accessibility

The `dropdown.js` script automatically handles keyboard navigation (Escape key to close) and focus management. Ensure your toggle button has a clear, descriptive label.

Placement

The dropdown menu is positioned absolutely relative to its parent `.dropdown` container. Ensure this container is placed appropriately in your layout to avoid clipping issues.

Use Dividers

Use `

  • ` to group related actions within a long dropdown menu, improving scannability and organization.
