CSS Structure

The steps component uses semantic HTML with AXIOM01 design tokens:

<div class="steps">
  <span data-active="true">
    <i>1</i>
    <span>Step One</span>
  </span>
  <span>
    <i>2</i>
    <span>Step Two</span>
  </span>
</div>

JavaScript API

Use the window.Axiom.setActiveStep() method to control steps programmatically:

// Set step 2 as active
const stepsContainer = document.querySelector('.steps');
window.Axiom.setActiveStep(stepsContainer, 1); // 0-based index

// Listen for step changes
stepsContainer.addEventListener('axiomStepChange', (e) => {
  console.log('Active step:', e.detail.activeIndex);
});

Data Attributes

  • data-active="true" - Marks the currently active step
  • data-completed="true" - Marks a completed step (displays checkmark or completed state)

Design Tokens Used

  • --a-space-l - Gap between steps
  • --a-space-s - Gap between icon and label
  • --a-space-xl - Step circle size
  • --a-font-weight-medium - Step label weight
  • --a-color-primary - Active step color
  • --a-color-success - Completed step color
  • --a-transition-base - Smooth state transitions
