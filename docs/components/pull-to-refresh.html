Pull To Refresh

Refresh content with a pull-to-refresh gesture.

Basic Usage

Drag down from the top edge to request a content refresh.

Pull to refresh

Feed

Drag down and release after the threshold to trigger refresh feedback.

The indicator uses transform-based motion for subtle animation.

Refresh events dispatch as axiom:refresh for app integration.

Required Markup Contract

  • Use data-component="pullrefresh" on the root container.
  • Include an indicator element (for example output) with aria-live.
  • Provide a refresh surface marked with data-role="surface".
