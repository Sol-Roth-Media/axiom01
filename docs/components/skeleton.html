Skeleton Component

Loading placeholder components that mimic the layout of content while it loads, providing better perceived performance.

Basic Skeleton

Simple text skeletons for loading states.

<div class="skeleton" role="status" aria-label="Loading content">
  <div class="text"></div>
  <div class="text short"></div>
  <div class="text medium"></div>
</div>

Card Skeleton

Skeleton placeholder for card components with image and text content.

<article class="card skeleton" role="status" aria-label="Loading article">
  <div class="image"></div>
  <main>
    <div class="text"></div>
    <div class="text short"></div>
    <div class="text medium"></div>
  </main>
</article>

Profile Skeleton

User profile skeleton with avatar and bio information.

<section class="skeleton profile" role="status" aria-label="Loading profile">
  <div class="avatar"></div>
  <div>
    <div class="text"></div>
    <div class="text short"></div>
  </div>
</section>

List Skeleton

Skeleton for list items and repeated content structures.

<div class="skeleton list" role="status" aria-label="Loading list">
  <article>
    <div class="avatar"></div>
    <div>
      <div class="text"></div>
      <div class="text short"></div>
    </div>
  </article>
</div>

Accessibility Features

  • Screen Reader Support: Proper ARIA labels and status roles
  • Loading States: Clear indication that content is loading
  • Semantic Structure: Uses appropriate HTML elements
  • Animation Control: Respects reduced motion preferences
