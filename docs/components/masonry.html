Masonry Layout

Axiom01's masonry layout uses CSS Grid to create beautiful, responsive multi-column layouts with variable-sized items. Perfect for image galleries, portfolios, and content grids.

HTML Structure

<div class="masonry-grid">
    <div class="masonry-item">
        <div class="masonry-content">🎨</div>
        <div class="masonry-label">Item 1</div>
    </div>
    <div class="masonry-item wide">
        <div class="masonry-content">🖼️</div>
        <div class="masonry-label">Wide Item</div>
    </div>
    <div class="masonry-item tall">
        <!-- Tall item spans 2 rows -->
    </div>
</div>

Features

  • Responsive Grid - Auto-fills columns based on space
  • Variable Sizes - Wide and tall item modifiers
  • Gap Control - Consistent spacing between items
  • Auto-Rows - Uniform row height management
  • Hover Effects - Smooth transitions on interaction
  • Mobile Responsive - Single column on small screens

Use Cases

  • Image Gallery - Portfolio and photography showcase
  • Product Grid - E-commerce product listings
  • Content Cards - Mixed-size content display
  • Dashboard - Widget and panel layout
  • Portfolio - Project showcase with variety
  • Blog Archive - Featured and regular posts

Accessibility

  • Include alt text for images
  • Use semantic grid item containers
  • Ensure keyboard navigation works
  • Maintain focus visibility on all items
  • Provide labels for unlabeled content
  • Test with screen readers

Best Practices

  • Consistent Spacing - Uniform gaps between items
  • Visual Hierarchy - Vary sizes purposefully
  • Content Strategy - Logical item ordering
  • Image Optimization - Properly sized images
  • Responsive Design - Test on all breakpoints
  • Performance - Lazy load images if needed
