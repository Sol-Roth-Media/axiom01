Swiping Cards

Card-swiping interaction using pointer events. Semantic <article> cards with CSS transitions.

Swipe Cards

<div class="swipecards" aria-label="Card stack">
  <article>
    <h3>Person Name</h3>
    <p>Role · Location</p>
    <p>Short description.</p>
  </article>
</div>
<div class="actions">
  <button aria-label="Skip">← Skip</button>
  <button class="primary" aria-label="Like">Like →</button>
</div>

Accessibility

  • Always provide button alternatives to swipe gestures
  • Announce card changes with aria-live="polite"
  • Ensure keyboard users can activate button alternatives
