Features

  • Multiple Sizes - Small, medium, large
  • Initials - User name display
  • Images - Profile pictures
  • Badges - Status indicators
  • Groups - Multiple avatars
  • Hover Effects - Interactive states

Use Cases

  • User Profiles - Account representation
  • Team Display - Member listings
  • Comments - Author identification
  • Notifications - User actions
  • Mentions - @ user tags
  • Chat - Message senders

Accessibility

  • Include alt text for images
  • Use initials for text avatars
  • Sufficient contrast on background
  • Provide user name in title/aria-label
  • Support screen readers
  • Meaningful status indicators

Best Practices

  • Consistent Sizing - Use standard sizes
  • Clear Initials - Use first letters
  • Quality Images - Crisp profile pics
  • Color Variety - Different gradients per user
  • Group Overlap - Visual connection
  • Loading State - Show placeholder
