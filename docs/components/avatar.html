Avatar Component

Avatar Component

Axiom01's avatar component is used to display a user's profile picture, initials, or a generic icon. It comes in various sizes and can be easily customized to fit your application's design.

Customizing Avatars

You can easily customize the background and text color of avatars using CSS variables or by adding custom classes that override the default styles.

.my-custom-avatar {
background-color: var(--a-color-secondary);
color: var(--a-color-on-secondary);
}

When to Use Avatar

Do Use

  • Use this component for its intended purpose
  • Follow the semantic structure
  • Implement proper accessibility
  • Respect responsive design principles
  • Use design tokens for customization

Don't Use

  • Don't modify core structure unnecessarily
  • Avoid hardcoding styles or colors
  • Don't skip accessibility features
  • Avoid responsive design gotchas
  • Don't duplicate when a component exists

Consider

  • Your specific use case and context
  • User accessibility needs (WCAG AAA)
  • Dark mode and theme variations
  • Mobile and desktop experiences
  • Performance optimization
