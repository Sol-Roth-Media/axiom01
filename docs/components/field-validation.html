Field Validation

Semantic form field wrapper with validation states using data attributes for error display, focus states, and validation feedback.

Basic Field

<div class="field">
    <label for="name">Full Name</label>
    <input type="text" id="name" placeholder="Enter your full name">
</div>

Field with Different Input Types

Email Input

Password Input

Textarea

Select Dropdown

<!-- Email field -->
<div class="field">
    <label for="email">Email Address</label>
    <input type="email" id="email" placeholder="user@example.com">
</div>

<!-- Textarea field -->
<div class="field">
    <label for="message">Message</label>
    <textarea id="message" rows="4"></textarea>
</div>

<!-- Select field -->
<div class="field">
    <label for="country">Country</label>
    <select id="country">
        <option value="">Select...</option>
        <option value="us">United States</option>
    </select>
</div>

Invalid Field with Error State

Please enter a valid email address 
<!-- Invalid field state -->
<div class="field" data-invalid="true">
    <label for="invalid-email">Email Address</label>
    <input type="email" id="invalid-email" value="not-an-email" aria-invalid="true" aria-describedby="email-error">
    <div id="email-error" role="alert" style="color: var(--a-color-error); font-size: 0.875rem; margin-top: var(--a-space-s); display: flex; align-items: center; gap: var(--a-space-s);">
        <span class="axicon render" data-name="AlertCircle"></span>
        <span>Please enter a valid email address</span>
    </div>
</div>

Valid Field with Success State

Email is valid 
<!-- Valid field state -->
<div class="field" data-valid="true">
    <label for="valid-email">Email Address</label>
    <input type="email" id="valid-email" value="user@example.com" aria-describedby="email-success">
    <div id="email-success" style="color: var(--a-color-success); font-size: 0.875rem; margin-top: var(--a-space-s); display: flex; align-items: center; gap: var(--a-space-s);">
        <span class="axicon render" data-name="Check"></span>
        <span>Email is valid</span>
    </div>
</div>
