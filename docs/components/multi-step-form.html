Multi-Step Form

Axiom01's multi-step form component breaks complex forms into manageable sections with progress tracking. Features step indicators, validation, and navigation controls.

HTML Structure

<form class="wizard">
    <nav>
        <ol>
            <li data-active="true">
                <span>1</span>
                <small>Step 1</small>
            </li>
        </ol>
    </nav>
    
    <main>
        <section data-active="true">
            <h2>Title</h2>
            <label>Field</label>
            <input type="text">
        </section>
    </main>
    
    <footer>
        <button type="button" class="secondary">Previous</button>
        <button type="button" class="primary">Next</button>
    </footer>
</form>

Features

  • Progress Indicator - Visual step tracking
  • Step Navigation - Previous/Next controls
  • Form Validation - Validate before advancing
  • Semantic HTML - Proper form structure
  • Success State - Completion confirmation
  • Responsive - Vertical progress on mobile

Use Cases

  • Account Registration - Multi-step signup flow
  • Checkout - E-commerce purchase process
  • Onboarding - Setup wizard for new users
  • Surveys - Multi-page questionnaires
  • Applications - Job or grant applications
  • Configuration - Complex app setup

Accessibility

  • Semantic form and section elements
  • Proper labels on all inputs
  • Data attributes for state management
  • Keyboard navigation support
  • Accessibility-first HTML structure
  • WCAG 2.1 AA compliant

Best Practices

  • Clear Progress - Show which step user is on
  • Logical Grouping - Related fields on same step
  • Validation - Validate before advancing
  • Save Progress - Remember entered data
  • Error Messages - Clear, actionable feedback
  • Responsive Design - Works on all screen sizes
