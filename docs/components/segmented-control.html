Segmented Control

Axiom01's segmented control component provides a modern alternative to tab navigation and radio buttons. Users can toggle between multiple related options with a single interaction.

Live Examples

View Mode Selector

List View

Display items in a detailed list format with all information visible. Ideal for browsing and detailed inspection of each item.

Grid View

Display items in a grid layout for visual browsing. Perfect for galleries, products, or media collections.

Compact View

Display items in a condensed format with minimal information. Ideal for quick scanning of many items.

HTML Structure

<div class="segmented-control" role="tablist">
    <button class="segment-button active" 
            data-panel="list" 
            role="tab" 
            aria-selected="true">
        List
    </button>
    <button class="segment-button" 
            data-panel="grid" 
            role="tab" 
            aria-selected="false">
        Grid
    </button>
</div>

<div class="content-panels">
    <div class="content-panel active" id="list">
        Content for list view
    </div>
    <div class="content-panel" id="grid">
        Content for grid view
    </div>
</div>

Features

  • Multiple Options - Support 2-5 segments
  • Visual Feedback - Clear active state indication
  • Icon Support - Include icons with labels
  • Content Switching - Dynamic panel display
  • Smooth Transitions - Fade animation on switch
  • Accessible - Full ARIA tab support

Use Cases

  • View Modes - List/grid/compact toggles
  • Filters - All/active/archived selection
  • Time Range - Day/week/month/year views
  • Sorting - Different sort order options
  • Display Options - Detailed/summary/icon views
  • Form Steps - Step-by-step form navigation

Accessibility

  • Use role="tablist" on container
  • Use role="tab" on buttons
  • Update aria-selected when switching
  • Support keyboard navigation (Arrow keys, Enter)
  • Announce content changes to screen readers
  • Provide visible focus indicators

Best Practices

  • Clear Labels - Descriptive option names
  • Visual Hierarchy - Emphasize active segment
  • Limited Options - Keep to 2-5 choices max
  • Consistent Width - Equal segment sizes
  • Icon + Text - Use both for clarity
  • Smooth Transitions - Fade between panels
