Segmented Control Component

Segmented Controls provide a way to select one item from a set of mutually exclusive options. Use them for filters, view modes, or content tabs. Built with semantic HTML and full keyboard accessibility.

Basic Example

<div class="segmented-control" role="group" aria-label="View options">
  <button aria-pressed="true">List</button>
  <button aria-pressed="false">Grid</button>
  <button aria-pressed="false">Map</button>
</div>

With Icons

<div class="segmented-control" role="group" aria-label="Sort options">
  <button aria-pressed="true"><i class="fas fa-arrow-down"></i> Newest</button>
  <button aria-pressed="false"><i class="fas fa-arrow-up"></i> Oldest</button>
  <button aria-pressed="false"><i class="fas fa-star"></i> Popular</button>
</div>

Full Width

<div class="segmented-control full-width" role="group" aria-label="Time period">
  <button aria-pressed="true">Today</button>
  <button aria-pressed="false">Week</button>
  <button aria-pressed="false">Month</button>
  <button aria-pressed="false">Year</button>
</div>

Features

Accessible

Uses aria-pressed for semantic screen reader support

Keyboard Nav

Tab and arrow keys for navigation, Enter to select

Icon Support

Works seamlessly with Font Awesome and other icon sets

Responsive

Stacks on mobile, stays compact on desktop

Themeable

Fully respects all Axiom01 design tokens

Semantic HTML

Pure CSS, no JavaScript required

Use Cases

  • View mode toggles (list/grid/map)
  • Sorting and filtering options
  • Time period selection
  • Theme or display preferences
  • Tab-like content switching
  • Exclusive option selection in forms

JavaScript Integration

For state management and switching content, add JavaScript to handle aria-pressed updates and content visibility. Example with vanilla JS:

document.querySelectorAll('.segmented-control button').forEach(btn => {
  btn.addEventListener('click', () => {
    // Update pressed state
    btn.parentElement.querySelectorAll('button').forEach(b => {
      b.setAttribute('aria-pressed', 'false');
    });
    btn.setAttribute('aria-pressed', 'true');
    
    // Emit event or update content
    const value = btn.textContent;
    console.log('Selected:', value);
  });
});

Browser Support

All modern browsers (Chrome, Firefox, Safari, Edge). No CSS preprocessor or JavaScript required for basic functionality.

When to Use Segmented Control

Do Use

  • Use this component for its intended purpose
  • Follow the semantic structure
  • Implement proper accessibility
  • Respect responsive design principles
  • Use design tokens for customization

Don't Use

  • Don't modify core structure unnecessarily
  • Avoid hardcoding styles or colors
  • Don't skip accessibility features
  • Avoid responsive design gotchas
  • Don't duplicate when a component exists

Consider

  • Your specific use case and context
  • User accessibility needs (WCAG AAA)
  • Dark mode and theme variations
  • Mobile and desktop experiences
  • Performance optimization
