Features

  • Product Cards - Clean, professional product display
  • Price Display - Original, current, and discount pricing
  • Rating System - 5-star ratings with review counts
  • Product Badges - NEW, HOT, SALE indicators
  • Wishlist - Save favorites for later
  • Responsive Grid - Auto-scales on all devices

Use Cases

  • E-Commerce Store - Complete product catalog display
  • Product Listing - Category and search results
  • Product Gallery - Featured products showcase
  • Shopping Cart - Selected items for purchase
  • Wishlist/Favorites - Saved items for later
  • Deal/Sale Page - Limited-time offers display

Accessibility

  • Use semantic HTML for product structure
  • Include aria-label on wishlist buttons
  • Provide alt text for product images
  • Support keyboard navigation through products
  • Use color + text for discounts (not just color)
  • Ensure rating indicators are understandable

Responsive Behavior

This component automatically adapts to different screen sizes:

Mobile (< 768px)

On mobile devices:

  • Optimized for small screens
  • Touch-friendly spacing maintained (44px minimum)
  • Text wraps naturally without overflow
  • Content density optimized for readability

Desktop (1024px+)

On desktop displays:

  • Enhanced visual features
  • Hover states and transitions
  • Full keyboard navigation support
  • Optimal spacing and sizing

Framework Integration

Responsive behavior is automatic:
  • Framework handles media queries
  • No custom CSS required
  • Mobile-first approach applied
  • Accessibility maintained at all sizes

Best Practices

  • Clear Pricing - Show original and current price clearly
  • Rating Display - Include star rating and review count
  • Product Images - High-quality visuals or placeholders
  • CTA Buttons - Clear "Add to Cart" call-to-action
  • Responsive Grid - Adapt to all screen sizes
  • Hover Effects - Subtle elevation/shadow on hover
