Video Player Component

A fully-featured HTML5 video player with custom controls, playback speed selection, volume control, and fullscreen support. Built with Axiom01 styling and keyboard accessibility.

Features

  • HTML5 video with adaptive bitrate support
  • Custom play/pause, progress, volume controls
  • Playback speed adjustment (0.5x to 2x)
  • Fullscreen mode
  • Time display with keyboard shortcuts
  • Fully keyboard accessible
  • Responsive design

Basic Usage

HTML Structure

<div data-component="video-player"
     data-src="path/to/video.mp4"
     data-poster="path/to/poster.jpg"
     data-type="video/mp4">
</div>

Attributes

Attribute Type Description
data-src URL Video source URL (required)
data-poster URL Thumbnail image shown before playback
data-type MIME type Default: "video/mp4"
data-autoplay Boolean Auto-start playback
data-loop Boolean Loop video when finished

JavaScript API

const playerEl = document.querySelector('[data-component="video-player"]');
const api = window.componentInstances.get(playerEl);

// Playback control
api.play();
api.pause();
api.seek(60); // Seek to 60 seconds

// Source control
api.setSource("new-video.mp4", "video/mp4");

// Audio control
api.setVolume(0.5); // 0-1

// Playback rate
api.setPlaybackRate(1.5);

// Get info
console.log(api.getCurrentTime());
console.log(api.getDuration());

Keyboard Shortcuts

Key Action
Space / K Play / Pause
M Mute / Unmute
F Fullscreen
← / → Skip backward / forward 5 seconds
Click progress bar Seek to position
