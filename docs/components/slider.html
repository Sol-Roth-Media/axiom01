Slider

Axiom01's slider component provides a smooth range input for selecting values. Supports single and range sliders with keyboard navigation and accessibility.

Live Examples

HTML Structure

<div class="slider-label">
    <span>Volume</span>
    <span class="slider-value">50%</span>
</div>
<div class="slider-container">
    <input type="range" min="0" max="100" value="50">
</div>

Features

  • Range Input - Select value from min/max
  • Custom Styling - Professional appearance
  • Smooth Interaction - Responsive drag behavior
  • Value Display - Show current selection
  • Keyboard Support - Arrow key navigation
  • Hover Effects - Visual feedback on interaction

Use Cases

  • Volume Control - Audio/video volume
  • Brightness - Screen brightness adjustment
  • Price Range - Filter by price
  • Playback Speed - Video speed control
  • Settings - Configuration sliders
  • Filters - Search and filter ranges

Accessibility

  • Use semantic <input type="range">
  • Include aria-label for context
  • Support keyboard arrow keys
  • Display current value clearly
  • Provide min/max labels
  • Ensure sufficient touch target size

Best Practices

  • Clear Labels - Explain what is being adjusted
  • Show Value - Display current selection
  • Step Increments - Appropriate step values
  • Visual Feedback - Hover and active states
  • Touch Friendly - Large enough touch targets
  • Min/Max Info - Display range boundaries
