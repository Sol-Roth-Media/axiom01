Audio Player Component

A beautiful, compact HTML5 audio player with album art, metadata display, volume control, and playback speed adjustment. Perfect for music streaming, podcasts, and audio content.

Features

  • Album artwork display
  • Track title and artist information
  • Play/pause control
  • Progress bar with seeking
  • Volume control slider
  • Playback speed adjustment (0.5x to 2x)
  • Download button
  • Queue/playlist integration
  • Responsive layout

Basic Usage

HTML Structure

<div data-component="audio-player"
     data-src="path/to/audio.mp3"
     data-title="Track Title"
     data-artist="Artist Name"
     data-cover="path/to/cover.jpg">
</div>

Attributes

Attribute Type Description
data-src URL Audio source URL (required)
data-title String Track title to display
data-artist String Artist or creator name
data-cover URL Album/cover artwork image
data-autoplay Boolean Auto-start playback

JavaScript API

const playerEl = document.querySelector('[data-component="audio-player"]');
const api = window.componentInstances.get(playerEl);

// Playback control
api.play();
api.pause();
api.seek(45); // Seek to 45 seconds

// Source control
api.setSource("new-track.mp3", "Track Title", "Artist Name");

// Volume control (0-1)
api.setVolume(0.8);

// Playback rate
api.setPlaybackRate(1.5);

// Get info
console.log(api.getCurrentTime());
console.log(api.getDuration());

Events

Event Detail
audio-changed { src, title, artist }

Best Practices

  • Audio Formats: Provide MP3 for broad compatibility, WebM/Opus for modern browsers
  • Cover Art: Use square images (minimum 80x80px, optimal 300x300px)
  • Metadata: Include clear artist and title for accessibility
  • Performance: Preload metadata with data-preload="metadata"

Browser Support

  • Chrome 90+
  • Firefox 88+
  • Safari 14+
  • Edge 90+
  • Mobile browsers (iOS 10+, Android 5+)

Required Markup Contract

  • Use data-component="audio-player" on the root player container.
  • Provide a valid audio URL via data-src.
  • Provide metadata fields data-title and data-artist; optional cover via data-cover.
