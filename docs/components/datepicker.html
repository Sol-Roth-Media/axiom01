Datepicker

Semantic datepicker component with calendar grid interface. Supports single date selection, date ranges, and HTML5 date inputs with full keyboard navigation and accessibility.

HTML Structure

<!-- Simple date input -->
<div class="field" data-type="date">
    <label for="date">Select Date</label>
    <input type="date" id="date">
</div>

<!-- Date range -->
<div class="field" data-type="date">
    <label>Date Range</label>
    <div class="date-range">
        <div class="field" data-type="date">
            <label for="start">Start Date</label>
            <input type="date" id="start">
        </div>
        <div class="field" data-type="date">
            <label for="end">End Date</label>
            <input type="date" id="end">
        </div>
    </div>
</div>

<!-- Calendar grid -->
<div role="grid" aria-label="Calendar">
    <div role="columnheader">Sun</div>
    <button role="gridcell" data-today="true">12</button>
    <button role="gridcell" data-selected="true">13</button>
</div>

Data Attributes

  • `data-type="date"` - Container for date field
  • `data-today="true"` - Marks today's date in calendar
  • `data-selected="true"` - Marks selected date
  • `data-other-month="true"` - Days from adjacent months

ARIA Attributes

  • `role="grid"` - Calendar container
  • `role="columnheader"` - Day of week headers
  • `role="gridcell"` - Individual calendar dates
  • `tabindex` - Keyboard navigation in calendar

Key Features

  • HTML5 Date Input - Native browser support with fallback
  • Single Date Selection - Pick one date easily
  • Date Range - Select start and end dates
  • Calendar Grid - Visual calendar interface
  • Today Highlight - Current date marked with data-today
  • Month Navigation - Navigate between months
  • Keyboard Navigation - Full keyboard support

Use Cases

  • Booking Systems - Hotel, flight, appointment booking
  • Event Registration - Conference or event date selection
  • Date Range Filters - Analytics and reporting date ranges
  • Birthday Selection - User profile and account forms
  • Project Timelines - Project start and end dates
  • Leave Request - Time off and vacation requests

Best Practices

  • Use Native Input - Let browsers handle date picking when possible
  • Consistent Format - Always use YYYY-MM-DD format internally
  • Today Button - Quick access to today's date
  • Range Validation - Ensure end date is after start date
  • Full Keyboard Support - Navigate calendar with arrows
  • Mobile Friendly - Native date picker on mobile devices
