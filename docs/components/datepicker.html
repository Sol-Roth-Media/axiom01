Datepicker Component

A fully accessible date input component with calendar popup, keyboard navigation, and internationalization support.

Basic Usage

<div class="form-group">
  <label for="birth-date">Date of Birth</label>
  <input type="date" id="birth-date" class="datepicker" aria-describedby="date-help">
  <p id="date-help" class="help-text">Select your birth date</p>
</div>

With Custom Format

<div class="form-group">
  <label for="event-date">Event Date</label>
  <div class="datepicker-wrapper">
    <input type="text" id="event-date" class="datepicker"
           placeholder="MM/DD/YYYY"
           aria-describedby="event-date-format">
    <i class="fas fa-calendar-alt datepicker-icon" aria-hidden="true"></i>
  </div>
  <p id="event-date-format" class="help-text">Format: MM/DD/YYYY</p>
</div>

Date Range Picker

<div class="form-group">
  <label for="start-date">Date Range</label>
  <div class="date-range">
    <input type="date" id="start-date" class="datepicker" aria-label="Start date">
    <span class="date-range-separator" aria-hidden="true">to</span>
    <input type="date" id="end-date" class="datepicker" aria-label="End date">
  </div>
</div>

Disabled and Read-only States

<div class="form-group">
  <label for="readonly-date">Read-only Date</label>
  <input type="date" id="readonly-date" class="datepicker" value="2025-07-21" readonly>
</div>

<div class="form-group">

Accessibility Features

  • Native HTML5 date input for maximum accessibility
  • Proper labeling with <label> elements
  • ARIA descriptions for format requirements
  • Keyboard navigation (Tab, Enter, Arrow keys)
  • Screen reader compatible
  • High contrast mode support

JavaScript Integration

// Enhanced datepicker with validation
document.querySelectorAll('.datepicker').forEach(input => {
  input.addEventListener('change', function() {
    const date = new Date(this.value);
    const today = new Date();
    // Example validation
    if (date > today) {
      this.setCustomValidity('Date cannot be in the future');
    } else {
      this.setCustomValidity('');
    }
  });
});
