Stats / Metrics

Numerical data display component using the stat class. Clean, accessible metric cards for dashboards and overview pages.

Default Stats

<article class="stat"> <span class="stat-value">73</span> <span class="stat-label">Components</span> </article>

With Icon and Meta

<article class="stat">
  <span class="axicon render" data-name="Users"></span>
  <span class="stat-value">24,891</span>
  <span class="stat-label">Active Users</span>
  <p class="stat-meta">↑ 12% from last month</p>
</article>

Semantic Structure

ElementRoleUse
article.statRootContainer — use article for self-contained data
span.stat-valueNumberThe main metric — large, prominent
span.stat-labelLabelWhat the number represents
p.stat-metaContextTrend, timeframe, comparison
span.axiconIcon (optional)Place before value for visual category

Accessibility

  • Wrap in a <section aria-label="Key Metrics">
  • Each article.stat is self-contained — screen readers treat it independently
  • Include trend info in the visible stat-meta — don't use color alone
  • Consider aria-label on the value if the label follows: <span class="stat-value" aria-label="24891 active users">
