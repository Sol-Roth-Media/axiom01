Stats/Metrics Component

Display key performance indicators, statistics, and metrics with a clean, readable card layout perfect for dashboards and landing pages.

Overview

The Stats component is a semantic, grid-friendly card for displaying numerical metrics with labels and optional metadata. Perfect for KPIs, achievements, and data visualization.

Basic Usage

Simple Stat Card

42
Active Projects
<article class="stat">
    <div class="stat-value">42</div>
    <div class="stat-label">Active Projects</div>
</article>

With Metadata

$12,340
Monthly Revenue
↑ 12% from last month
<article class="stat">
    <div class="stat-value">$12,340</div>
    <div class="stat-label">Monthly Revenue</div>
    <div class="stat-meta">↑ 12% from last month</div>
</article>

Component Structure

Element Purpose Required
<article class="stat"> Container for the metric card Yes
.stat-value The main metric number or value Yes
.stat-label Label describing what the metric represents Yes
.stat-meta Optional metadata like trends or comparison No

Use Cases

  • Dashboard Metrics: Display KPIs and real-time data
  • Landing Pages: Show achievements and statistics to build credibility
  • Reports: Present summary statistics at a glance
  • Analytics Pages: Showcase performance metrics
  • About Pages: Highlight company achievements
  • Social Proof: Display user counts, satisfaction rates, etc.

Styling & Customization

/* Custom stat card colors */
.stat.success .stat-value {
    color: var(--a-color-success);
}

.stat.warning .stat-value {
    color: var(--a-color-warning);
}

.stat.error .stat-value {
    color: var(--a-color-error);
}

/* Adjust sizing */
.stat-value {
    font-size: 3rem; /* Larger values */
}

.stat-label {
    font-size: 1.2rem; /* Larger labels */
}

Grid Layouts

Use Axiom's grid system to create responsive stat layouts:

/* Responsive 2-column on mobile, 4-column on desktop */
<div class="grid">
    <article class="stat">...</article>
    <article class="stat">...</article>
    <article class="stat">...</article>
    <article class="stat">...</article>
</div>

/* Custom grid in CSS */
.stat-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: var(--a-space-large);
}

Accessibility

  • Semantic HTML: Uses <article> element for proper structure
  • Clear Labels: Each stat has a descriptive label
  • Text Contrast: High contrast values and labels
  • Icon Support: Icons should have descriptions or aria-labels
  • Responsive: Stacks properly on all screen sizes

Metadata Tips

  • Use ↑ or ↓ arrows for trending data
  • Include percentage change: ↑ 23% from last month
  • Show time period: ↓ 2% this quarter
  • Add comparison: vs. 325 last week
  • Keep metadata concise and scannable
