Media Component

Responsive media handling for images, videos, and audio with optimization and accessibility features.

Responsive Image

Responsive images with proper semantic structure and captions.

<figure class="media">
  <img src="image.jpg" alt="Descriptive alt text" class="responsive">
  <figcaption>Image caption providing context</figcaption>
</figure>

Image with Lazy Loading

Performance-optimized images with lazy loading and progressive enhancement.

<div class="media lazy">
  <img src="placeholder.jpg"
       data-src="high-res-image.jpg"
       alt="Descriptive alt text"
       loading="lazy">
</div>

Video Player

Accessible video player with captions and multiple format support.

<div class="media video">
  <video controls aria-label="Product demonstration video">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <track kind="captions" src="captions.vtt" srclang="en" label="English captions">
    <p>Your browser doesn't support HTML5 video. <a href="video.mp4">Download the video</a> instead.</p>
  </video>
</div>

  --media-border-radius: var(--a-border-radius);
  --media-shadow: var(--a-shadow-m);
  --gallery-gap: var(--a-space-m);

}

Accessibility Features

  • Alt Text: Descriptive alternative text for all images
  • Captions: Video and audio captions for hearing impaired users
  • Keyboard Controls: Full keyboard support for media players
  • Screen Reader Support: Proper ARIA labels and semantic markup
  • Fallback Content: Alternative content for unsupported media formats
