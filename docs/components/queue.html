Features

  • Queue Display - Visual list of waiting customers
  • Status Indicators - Serving, waiting, completed states
  • Position Numbers - Clear queue position display
  • Progress Bar - Visual queue progress indicator
  • Timeline View - Service history tracking
  • Customer Info - Name, ticket number, and details

Use Cases

  • Service Desks - Bank, DMV, post office queues
  • Restaurant Booking - Waiting list management
  • Customer Support - Help desk ticket queue
  • Healthcare - Patient waiting lists
  • Event Registration - Ticket redemption queues
  • Print Shop - Job queue and status tracking

Accessibility

  • Use semantic list structure for queue items
  • Clearly indicate serving vs waiting status
  • Use colors + text for status (not just color)
  • Include aria-label for status badges
  • Support keyboard navigation through queue
  • Provide text alternative for progress indicators

Best Practices

  • Clear Status - Make current service obvious
  • Position Tracking - Show customer's position in queue
  • Time Estimates - Display wait time if available
  • Real-Time Updates - Update as customers move through
  • Visual Hierarchy - Emphasize currently serving customer
  • History Tracking - Show completed services timeline
