Queue (Drag & Drop)

Reorderable lists with native drag-and-drop support. Perfect for task management, reordering form fields, and playlist management. Fully compatible with forms and includes animation.

Queue Component

The Queue component provides a semantic list (<ol class="queue"> or <ul class="queue">) that allows users to drag and reorder items. Each item can be removed, and the component automatically syncs with hidden form fields for form submission.

Basic Example

  1. Learn JavaScript
  2. Build a project
  3. Deploy to production
<ol class="queue" data-queue>
  <li>Learn JavaScript</li>
  <li>Build a project</li>
  <li>Deploy to production</li>
</ol>

With Form Integration

Link the queue to a form by adding data-form-field attribute. The component will automatically update a hidden input with the reordered values.

Reorder Your Priorities
  1. High Priority Task
  2. Medium Priority Task
  3. Low Priority Task
<form>
  <ol class="queue" data-form-field="priorities">
    <li data-value="1">High Priority Task</li>
    <li data-value="2">Medium Priority Task</li>
    <li data-value="3">Low Priority Task</li>
  </ol>
  <input type="hidden" name="priorities" value="1,2,3">
  <button type="submit">Submit Order</button>
</form>

Features

Drag & Drop

Native HTML5 drag-and-drop API with smooth visual feedback

Form Sync

Automatically updates hidden input fields when items are reordered

Item Removal

Built-in remove buttons with smooth animations

Mobile Ready

Touch-friendly with haptic feedback support

Animated

Smooth transitions and staggered animations on load

Accessible

Full keyboard support and ARIA labels

Use Cases

  • Task management and todo lists
  • Playlist and queue management
  • Form field reordering
  • Priority and ranking systems
  • Content ordering in CMSs
  • Menu item reorganization

JavaScript Required

The Queue component requires js/axiom.js to be included. It automatically initializes all elements with class="queue".

Browser Support

Chrome, Firefox, Safari, Edge, and all modern browsers supporting HTML5 Drag & Drop API and ES6+.
