Accordion

Semantic accordion component for organizing expandable content sections. Single `.accordion` class with semantic HTML and ARIA attributes.

Live Example

<div class="accordion" role="region">
				<button class="accordion-button" aria-expanded="true" aria-controls="panel-1">
				<span class="axicon render" data-name="ChevronDown"></span>
				Section 1
				</button>
				<div id="panel-1" class="accordion-panel">
				<p>Content here...</p>
				</div>

				<button class="accordion-button" aria-expanded="false" aria-controls="panel-2">
				<span class="axicon render" data-name="ChevronDown"></span>
				Section 2
				</button>
				<div id="panel-2" class="accordion-panel" hidden>
				<p>Content here...</p>
				</div>
				</div>

Markup Structure

Accordion uses semantic buttons and divs with ARIA attributes:

<div class="accordion" role="region">
				<!-- Button to toggle panel -->
				<button class="accordion-button"
				aria-expanded="true|false"
				aria-controls="panel-id">
				<span class="axicon render" data-name="ChevronDown"></span>
				Section Title
				</button>

				<!-- Content panel -->
				<div id="panel-id" class="accordion-panel">
				<p>Panel content...</p>
				</div>

				<!-- Repeat button/panel pairs -->
				</div>

Key Features

  • Semantic Structure - Uses `
  • Single Class - `.accordion` container, `.accordion-button` and `.accordion-panel`
  • ARIA Attributes - `aria-expanded`, `aria-controls` for accessibility
  • Icon Rotation - Chevron rotates to show expanded state
  • Keyboard Navigation - Tab to buttons, Space/Enter to toggle
  • Hidden State - Uses native `hidden` attribute
  • No BEM Classes - No `.accordion-header`, `.accordion-content`

JavaScript Implementation

Simple JavaScript for accordion functionality:

// Accordion toggle functionality
				document.querySelectorAll('.accordion-button').forEach(button => {
				button.addEventListener('click', function() {
				const expanded = this.getAttribute('aria-expanded') === 'true';
				const panelId = this.getAttribute('aria-controls');
				const panel = document.getElementById(panelId);

				// Toggle button state
				this.setAttribute('aria-expanded', !expanded);

				// Toggle panel visibility
				if (expanded) {
				panel.setAttribute('hidden', '');
				} else {
				panel.removeAttribute('hidden');
				}

				// Optional: Close other panels (single open at a time)
				// document.querySelectorAll('.accordion-button').forEach(btn => {
				//     if (btn !== this) {
				//         btn.setAttribute('aria-expanded', 'false');
				//         document.getElementById(btn.getAttribute('aria-controls')).setAttribute('hidden', '');
				//     }
				// });
				});
				});

ARIA Attributes

  • `role="region"` - Container identifies as a region
  • `aria-expanded="true|false"` - Shows if section is expanded
  • `aria-controls="panel-id"` - Links button to controlled panel
  • `hidden` attribute - Hides collapsed panels semantically

Accessibility

  • Use `
  • Include `aria-expanded` to show state
  • Link buttons to panels with `aria-controls`
  • Use `hidden` attribute for collapsed content
  • Support keyboard: Tab navigation, Space/Enter to toggle
  • Icon rotates to show expanded state
  • Sufficient color contrast
  • Test with screen readers
