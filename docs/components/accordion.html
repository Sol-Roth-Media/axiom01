What is Axiom01?

Axiom01 is a comprehensive, semantic web framework built with HTML, CSS, and SVG. It provides 74+ production-ready components designed with accessibility and responsive design at their core.

Is Axiom01 free?

Yes, Axiom01 is completely free and open-source. It's available on GitHub under an MIT license, allowing you to use it in both personal and commercial projects.

Does it support dark mode?

Yes! Axiom01 comes with full dark mode support built-in. Simply toggle between light and dark themes, and all components automatically adapt their colors.

How do I get started?

Check out the Overview & Quickstart guide. It covers installation via npm and CDN, plus simple examples to get you started in minutes.

HTML Structure

Accordions use native <details> and <summary> elements for built-in accessibility:

<div class="accordion">
    <details class="accordion-item">
        <summary class="accordion-header">
            <span>Section Title</span>
            <button aria-hidden="true">
                <span class="axicon render" data-name="Chevron-Down"></span>
            </button>
        </summary>
        <div class="accordion-body">
            <p>Content goes here...</p>
        </div>
    </details>
</div>

Features

  • Native Details Element: Uses HTML5 <details> for accessibility
  • No JavaScript Required: Works with native browser functionality
  • Icon Animation: Chevron rotates smoothly on expand/collapse
  • Smooth Transitions: Hover effects on headers
  • Flexible Content: Can contain any HTML content
  • Multiple Open Items: Multiple sections can be open simultaneously

Use Cases

  • FAQ Pages: Organize frequently asked questions
  • Documentation: Collapsible sections for better organization
  • Settings: Group related options in expandable sections
  • Product Details: Show specifications, reviews, or shipping info
  • Help Center: Organize help topics hierarchically

Accessibility

  • Built on native <details> element - no ARIA needed
  • Keyboard accessible - use Tab and Enter/Space to expand
  • Screen readers announce expanded/collapsed state automatically
  • Decorative icons use aria-hidden="true"
  • Clear visual indicator of open/closed state (icon rotation)

Best Practices

  • Short Titles: Keep section titles concise (ideally under 60 characters)
  • Consistent Styling: All accordion items should look the same
  • Clear Icons: Use chevron or arrow icons that rotate smoothly
  • Hover States: Make headers interactive with hover effects
  • Don't Nest Accordions: Avoid nesting accordions too deeply
