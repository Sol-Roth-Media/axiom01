Progress & Skeleton

Axiom01's progress and skeleton loader components provide visual feedback for ongoing operations and loading states. Essential for improved perceived performance.

Features

  • Determinate Progress - Show percentage complete
  • Indeterminate Progress - Animate while loading
  • Status Colors - Success, warning, error states
  • Skeleton Screens - Placeholder while loading
  • Smooth Animations - Fluid transitions
  • Accessible - aria-valuenow and aria-valuemax

Use Cases

  • File Operations - Upload, download progress
  • Loading States - Content being fetched
  • Installation - Setup progress tracking
  • Batch Operations - Process status display
  • Perceived Performance - Skeleton screens
  • Long Operations - Background task progress

Accessibility

  • Use role="progressbar" on determinate progress
  • Include aria-valuenow and aria-valuemax
  • Provide percentage text alongside bar
  • Use color + text for status indication
  • Announce completion to screen readers
  • Support high contrast mode

Best Practices

  • Show Progress - Always indicate ongoing operation
  • Accurate Estimates - Don't fake progress completion
  • Skeleton Screens - Better UX than spinners
  • Color Meaning - Use consistent color semantics
  • Clear Labels - Show what's loading
  • Completion Feedback - Signal when done
