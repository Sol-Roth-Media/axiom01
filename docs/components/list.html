Features

Staggered Animation

Each item appears with a slight delay for a cascading effect

Smooth Transitions

Ease-out timing function for natural, polished motion

Pure CSS

No JavaScript required—animations work instantly

Respects Preferences

Honors prefers-reduced-motion for accessibility

Theme Compatible

Works seamlessly with all Axiom01 themes

Responsive

Perfect on all screen sizes

Animation Details

  • Duration: 0.5s per item
  • Stagger delay: 0.1s between items
  • Timing function: ease-out
  • Effect: Fade-in + slide-up 20px
  • Max items: 5 (auto-delay after that)

Use Cases

  • Feature lists and benefits
  • Step-by-step instructions
  • Content feeds and articles
  • Team member or testimonial lists
  • Integration or partner lists
  • Pricing tiers or plan features

Browser Support

All modern browsers. Uses standard CSS animations with no prefixes required (Chrome, Firefox, Safari, Edge 88+).
