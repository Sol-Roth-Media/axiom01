Inputs & Forms

Axiom provides clean, accessible, and themeable styles for all standard form elements.

Text Inputs

Standard text inputs for single-line data entry. Use the `.form-group` container to properly space labels and inputs.

<div class="form-group">
    <label for="demo-text">Text Input</label>
    <input type="text" id="demo-text" placeholder="...">
</div>

Select & Textarea

For dropdown selections and multi-line text entry.

<div class="form-group">
    <label for="demo-select">Select Menu</label>
    <select id="demo-select">...</select>
</div>

<div class="form-group">
    <label for="demo-textarea">Textarea</label>
    <textarea id="demo-textarea"></textarea>
</div>
