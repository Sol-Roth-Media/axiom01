Drawer Component

A slide-out panel component for navigation, settings, or additional content. Provides smooth animations and proper focus management.

Basic Drawer

Left-side navigation drawer with semantic HTML structure.

Menu

  • Home
  • About
  • Services
  • Contact
<div class="drawer" id="main-drawer" aria-hidden="true">
  <div class="overlay" aria-hidden="true"></div>
  <nav role="navigation" aria-label="Main menu">
    <header>
      <h2>Menu</h2>
      <button class="close" aria-label="Close menu">
        <i class="fas fa-times" aria-hidden="true"></i>
      </button>
    </header>
    <div>
      <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
      </ul>
    </div>
  </nav>
</div>

<button class="trigger" data-drawer="main-drawer">
  <i class="fas fa-bars" aria-hidden="true"></i>
  <span>Open Menu</span>
</button>

Accessibility Features

  • Focus Management: Proper focus trapping within drawer
  • Keyboard Navigation: Escape key closes drawer
  • Screen Reader Support: ARIA labels and states
  • Semantic Elements: Proper use of nav, aside, header elements
  • Overlay Interaction: Click outside to close
