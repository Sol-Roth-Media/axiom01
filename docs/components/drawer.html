HTML Structure

<!-- Overlay backdrop -->
<div class="drawer-overlay" id="drawerOverlay"></div>

<!-- Drawer panel -->
<div class="drawer-panel" id="drawer" role="dialog" aria-modal="true" aria-labelledby="drawer-title">
    <div class="drawer-header">
        <h2 id="drawer-title">Drawer Title</h2>
        <button class="drawer-close-btn" aria-label="Close drawer">&times;</button>
    </div>
    
    <div class="drawer-body">
        <nav>
            <ul>
                <li><a href="#">Navigation Item</a></li>
            </ul>
        </nav>
    </div>
    
    <div class="drawer-footer">
        <p>Footer content</p>
    </div>
</div>

JavaScript Implementation

// Open drawer
document.getElementById('openDrawer').addEventListener('click', () => {
    document.getElementById('drawer').classList.add('open');
    document.getElementById('drawerOverlay').classList.add('open');
});

// Close drawer
document.getElementById('closeDrawer').addEventListener('click', () => {
    document.getElementById('drawer').classList.remove('open');
    document.getElementById('drawerOverlay').classList.remove('open');
});

// Close on overlay click
document.getElementById('drawerOverlay').addEventListener('click', () => {
    document.getElementById('drawer').classList.remove('open');
    document.getElementById('drawerOverlay').classList.remove('open');
});

Features

  • Smooth Animation - CSS transforms for hardware-accelerated sliding
  • Overlay Backdrop - Click outside to dismiss (optional)
  • Flexible Content - Header, body, and footer sections
  • Navigation Support - Built for navigation menus with icons
  • Scrollable Body - Long content automatically scrolls
  • Accessible - ARIA roles, keyboard support, focus management

Use Cases

  • Navigation Menu - Primary navigation on mobile devices
  • Settings Panel - User preferences and configuration options
  • Filters - Product filters for e-commerce or listings
  • Notifications - Notification center or inbox
  • Documentation - Table of contents or navigation sidebar
  • Chat - Chat history or conversation list

Accessibility

  • Use role="dialog" and aria-modal="true"
  • Include aria-labelledby pointing to the title
  • Provide close button with clear aria-label
  • Manage focus when opening and closing
  • Support Escape key to close (recommended)
  • Ensure keyboard navigation through links

Best Practices

  • Width - Keep drawer width 250-350px for optimal usability
  • Mobile First - Hide drawer on desktop or place inline
  • Content - Keep drawer content simple and focused
  • Animation - Use smooth 0.3s transition for better UX
  • Dismissal - Provide multiple close options (button, overlay click)
  • Z-index - Ensure drawer sits above all page content
