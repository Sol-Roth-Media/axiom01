Drawer

Modal slide-out drawer component for displaying additional content in a modal overlay.

Overview

The Drawer component is a semantic modal slide-out panel that overlays content. Built on native HTML5 `<dialog>` element with smooth slide animations and proper focus management.

Basic Usage

Simple Drawer

Drawer Title

Drawer content goes here. This modal slides in from the side.

 
<button type="button" data-drawer-open="drawer-demo">Open Drawer</button>

<dialog class="drawer" data-side="right">
    <header>
        <h2>Drawer Title</h2>
        <button type="button" data-drawer-close aria-label="Close drawer">✕</button>
    </header>
    <main>
        <p>Drawer content</p>
    </main>
</dialog>

Left-Aligned Drawer

Left Drawer

This drawer slides in from the left side.

 
<dialog class="drawer" data-side="left">
    <header>
        <h2>Left Drawer</h2>
        <button type="button" data-drawer-close>✕</button>
    </header>

    </main>
</dialog>

Structure

Element Purpose Required
<dialog class="drawer"> Native dialog element for drawer Yes
data-side Direction: "left" or "right" (default: "right") No
<header> Drawer header with title and close button Recommended
<main> Drawer content area Yes

Styling & Customization

/* Drawer dialog element */
dialog.drawer {
    background: var(--a-color-surface);
    border: 1px solid var(--a-color-outline);
    width: 100%;
    max-width: 400px;
    height: 100vh;
    margin: 0;
    padding: 0;
    animation: slideInRight 0.3s ease;
}

/* Left drawer variant */
dialog.drawer[data-side="left"] {
    animation: slideInLeft 0.3s ease;
}

/* Modal backdrop */
dialog.drawer::backdrop {
    background: rgba(0, 0, 0, 0.5);
    backdrop-filter: blur(5px);
}

/* Drawer header */
dialog.drawer header {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: var(--a-space-m);
    border-bottom: 1px solid var(--a-color-outline);
}

/* Drawer close button */
dialog.drawer header button {
    background: none;
    border: none;
    font-size: 1.5rem;
    cursor: pointer;
    color: var(--a-color-on-surface);
}

/* Animations */
@keyframes slideInRight {
    from { transform: translateX(100%); }
    to { transform: translateX(0); }
}

@keyframes slideInLeft {
    from { transform: translateX(-100%); }
    to { transform: translateX(0); }
}

Accessibility

  • Native Dialog: Uses semantic `<dialog>` element with built-in focus trapping
  • ARIA Labels: Close buttons have aria-label for screen readers
  • Keyboard Support: Esc key closes drawer automatically
  • Focus Management: Focus moves to drawer on open, returns on close
  • Backdrop: Click outside (on backdrop) can close drawer

Integration Examples

Drawer with Form

<button type="button" data-drawer-open="form-drawer">Edit Settings</button>

<dialog class="drawer" id="form-drawer" data-side="right">
    <header>
        <h2>Edit Settings</h2>
        <button type="button" data-drawer-close>✕</button>
    </header>
    <main>
        <form>
            <div class="field">
                <label for="name">Name</label>
                <input id="name" type="text" />
            </div>
            <div class="field">
                <label for="email">Email</label>
                <input id="email" type="email" />
            </div>
            <button type="submit">Save</button>
        </form>
    </main>
</dialog>

Drawer with Close on Click

// Close drawer when clicking backdrop
document.querySelectorAll('dialog.drawer').forEach(drawer => {
    drawer.addEventListener('click', (e) => {
        if (e.target === drawer) {
            drawer.close();
        }
    });
});

Use Cases

  • Filters: Slide-out filter panels for lists and tables
  • Settings: Configuration panels and user preferences
  • Forms: Create/edit forms in modal drawer
  • Navigation: Mobile-friendly navigation drawer
  • Details: Show detailed information without full page load
  • Shopping Cart: Shopping cart preview on e-commerce sites
