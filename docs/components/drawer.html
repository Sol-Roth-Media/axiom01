Drawer

Slide-in panel using the native <dialog> element. Full keyboard trap, focus management, and backdrop built in.

Open a Drawer

<button class="primary" onclick="document.getElementById('my-drawer').showModal()">
  Open Drawer
</button>

<dialog class="drawer" id="my-drawer">
  <header>
    <h2>Drawer Title</h2>
    <button aria-label="Close drawer"
            onclick="document.getElementById('my-drawer').close()">
      <span class="axicon render" data-name="X"></span>
    </button>
  </header>
  <main>
    <p>Content here.</p>
  </main>
</dialog>

Left Side Drawer

<dialog class="drawer" data-side="left" id="left-drawer">
  ...
</dialog>

Why Native <dialog>?

  • Built-in keyboard trap — focus stays inside until closed
  • Escape key closes it automatically
  • Automatic ::backdrop for overlay
  • No JavaScript needed for basic open/close — use .showModal() and .close()
  • Proper ARIA: role="dialog" and focus management are native
