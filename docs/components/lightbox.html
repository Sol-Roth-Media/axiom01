HTML Structure

Gallery Links

<a href="#" class="lightbox-trigger" data-image="1" role="button" aria-label="View image">
  <img src="thumbnail.jpg" alt="Image description">
</a>

Lightbox Dialog

<dialog class="lightbox" data-lightbox="gallery-id">
  <div class="lightbox-content">
    <img src="full-image.jpg" alt="Image">
    <figcaption>Optional caption</figcaption>
  </div>
  <button class="lightbox-close" aria-label="Close">
    <span class="axicon" data-name="Close"></span>
  </button>
  <button class="lightbox-control prev" aria-label="Previous">
    <span class="axicon" data-name="ChevronLeft"></span>
  </button>
  <button class="lightbox-control next" aria-label="Next">
    <span class="axicon" data-name="ChevronRight"></span>
  </button>
</dialog>

With Caption

<dialog class="lightbox">
  <div class="lightbox-content">
    <figure>
      <img src="image.jpg" alt="Description">
      <figcaption>Image title and description</figcaption>
    </figure>
  </div>
  <!-- controls -->
</dialog>

CSS Classes

.lightbox

Native <dialog> element with fullscreen modal styling.

.lightbox-trigger

Clickable link to open lightbox modal.

.lightbox-content

Container for image and optional caption.

.lightbox-close

Close button (top-right corner).

.lightbox-control

Previous/next navigation buttons.

data-lightbox="id"

Groups images for gallery navigation.

JavaScript API

Open Lightbox

const lightbox = document.querySelector('.lightbox');

// Open dialog
lightbox.showModal();

// Close dialog
lightbox.close();

Listen for Events

const lightbox = document.querySelector('.lightbox');

lightbox.addEventListener('close', () => {
  console.log('Lightbox closed');
});

lightbox.addEventListener('cancel', () => {
  console.log('Lightbox cancelled');
});

Gallery Navigation

const nextBtn = document.querySelector('.lightbox-control.next');
const prevBtn = document.querySelector('.lightbox-control.prev');
let currentIndex = 0;

nextBtn.addEventListener('click', () => {
  currentIndex++;
  loadImage(currentIndex);
});

prevBtn.addEventListener('click', () => {
  currentIndex--;
  loadImage(currentIndex);
});

Accessibility

Best Practices

  • Use native <dialog> element
  • Provide aria-label on all controls
  • Include image alt text
  • Support keyboard navigation (arrow keys, escape)
  • Maintain focus management (trap in modal)
  • Announce gallery position (image 1 of 5)
  • Support screen readers for captions
  • Test with NVDA, JAWS, VoiceOver

AXIOM01 Philosophy

Why Use Native Dialog

Lightboxes embody AXIOM01 principles:

  • Semantic: Native <dialog> indicates modal context
  • Accessible: Browser handles focus trap and keyboard events
  • Zero Dependencies: No modal library needed
  • Responsive: Scales to any screen size
  • Performant: Hardware-accelerated animations
  • Progressive: Works without JavaScript

Avoid custom modal implementations that break accessibility. Use the native dialog element.
