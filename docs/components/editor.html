JavaScript Implementation

function setupEditor(containerId, contentId) {
    const toolbar = document.querySelectorAll(
        `#${containerId} .toolbar-button`
    );
    const content = document.getElementById(contentId);
    
    // Toolbar button click handlers
    toolbar.forEach(button => {
        button.addEventListener('click', (e) => {
            e.preventDefault();
            const format = button.dataset.format;
            const value = button.dataset.value;
            
            if (value) {
                document.execCommand(format, false, value);
            } else {
                document.execCommand(format, false, null);
            }
            
            content.focus();
            button.classList.toggle('active');
        });
    });
    
    // Update word/character count
    content.addEventListener('input', () => {
        const text = content.innerText || '';
        const words = text.trim().split(/\s+/).length;
        const chars = text.length;
        
        document.getElementById('wordCount').innerText = words;
        document.getElementById('charCount').innerText = chars;
    });
}

Features

  • Rich Text Formatting - Bold, italic, underline, strikethrough
  • List Support - Ordered and unordered lists
  • Block Elements - Headings, paragraphs, blockquotes
  • Toolbar Controls - One-click formatting buttons
  • Word/Character Count - Real-time statistics
  • Clear Formatting - Remove all formatting from selected text

Use Cases

  • Blog Publishing - Create and format blog posts
  • Note Taking - Rich notes with formatting
  • Email Composition - Formatted email bodies
  • Document Editing - Collaborative document creation
  • Content Management - CMS article editing
  • Comment Systems - Formatted user comments

Accessibility

  • Use role="textbox" on editable area
  • Include aria-label for content area
  • Provide keyboard shortcuts for all formatting
  • Show active state for applied formatting
  • Support Tab key for toolbar navigation
  • Announce character/word count updates

Best Practices

  • Save Drafts - Auto-save content periodically
  • Undo/Redo - Provide undo/redo functionality
  • Placeholder Text - Guide users with helpful hints
  • Keyboard Shortcuts - Ctrl+B for bold, Ctrl+I for italic
  • Spell Check - Built-in browser spell checking
  • Character Limit - Show limits if needed
