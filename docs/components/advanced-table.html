Advanced Table Component

Transform any HTML table into a powerful data management tool with built-in sorting, filtering, searching, and pagination. Perfect for displaying large datasets with full user control.

Overview

The Advanced Table component enhances standard HTML tables with professional data management features:

  • Click column headers to sort (ascending/descending)
  • Global search across all columns
  • Individual column filtering via dropdowns
  • Automatic pagination with customizable items per page
  • Clear visual indicators for sort direction
  • Results counter and navigation controls

      <tr>...</tr>
    </tbody>

Features

Sorting

Click any column header to sort. Click again to reverse direction. Visual indicators show current sort state.

Searching

Use the search box to filter results across all columns simultaneously. Results update in real-time.

Filtering

For columns with 20 or fewer unique values, filter dropdowns appear automatically above the table.

Pagination

Results are paginated with 10 items per page by default (customizable via API).

JavaScript API

const tableEl = document.querySelector('[data-component="advanced-table"]');
const api = window.componentInstances.get(tableEl);

// Set new data
api.setData([
  { col1: "Value 1", col2: "Value 2" },
  // ...
]);

// Get filtered data
const filtered = api.getFilteredData();

// Clear all filters and search
api.clearFilters();

// Set items per page
api.setItemsPerPage(25);

Best Practices

  • Column Headers: Keep header text concise and descriptive
  • Data Types: Table auto-detects numeric vs. string data for smarter sorting
  • Large Datasets: Consider chunking data or using backend filtering for 10,000+ rows
  • Accessibility: Use semantic table markup; component preserves all ARIA attributes
  • Styling: Striped rows and hover effects are built-in for better readability
