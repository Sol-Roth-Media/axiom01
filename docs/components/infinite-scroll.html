Infinite Scroll Component

Axiom01 provides the structural and visual foundation for implementing infinite scrolling. This pattern allows users to continuously load content as they scroll down a page, providing a seamless browsing experience without pagination.

Code Example

<div class="infinite-scroll-container" id="infinite-scroll-demo">
    <div class="infinite-scroll-item">Item 1</div>
    <div class="infinite-scroll-item">Item 2</div>
    <!-- ... more items ... -->
    <div class="loading-indicator" id="loading-indicator">Loading more items...</div>
</div>

<script>
    const scrollContainer = document.getElementById('infinite-scroll-demo');
    const loadingIndicator = document.getElementById('loading-indicator');
    let itemCount = 10;
    let isLoading = false;

    const loadMoreItems = () => {
        if (isLoading) return;
        isLoading = true;
        loadingIndicator.style.display = 'block';

        setTimeout(() => { // Simulate network request
            for (let i = 0; i < 5; i++) {
                itemCount++;
                const newItem = document.createElement('div');
                newItem.classList.add('infinite-scroll-item');
                newItem.textContent = `Item ${itemCount}: New content loaded.`;
                scrollContainer.insertBefore(newItem, loadingIndicator);
            }
            loadingIndicator.style.display = 'none';
            isLoading = false;
        }, 1000);
    };

    scrollContainer.addEventListener('scroll', () => {
        const { scrollTop, scrollHeight, clientHeight } = scrollContainer;
        if (scrollTop + clientHeight >= scrollHeight - 50 && !isLoading) { // 50px from bottom
            loadMoreItems();
        }
    });

    // Initial load if container is not full
    if (scrollContainer.scrollHeight <= scrollContainer.clientHeight) {
        loadMoreItems();
    }
</script>

Usage Guidelines

The .infinite-scroll-container class defines the scrollable area. Individual content blocks should use .infinite-scroll-item. A .loading-indicator can be used to provide feedback to the user while new content is being fetched.

Implementing infinite scroll requires JavaScript to detect when the user reaches the bottom of the scrollable area and then fetch and append new content. The example above provides a basic JavaScript implementation using a simulated delay.

Consider accessibility for users who might prefer traditional pagination or who may find infinite scrolling disorienting. Provide an alternative if necessary.
