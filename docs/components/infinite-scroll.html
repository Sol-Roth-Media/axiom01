Infinite Scroll

Axiom01's infinite scroll component automatically loads more content as users reach the bottom of a page. Perfect for feeds, lists, and galleries that continuously load new items.

JavaScript Implementation

let currentPage = 1;
let isLoading = false;
const scrollContainer = document.getElementById('scrollContainer');
const itemList = document.getElementById('itemList');
const loadingIndicator = document.getElementById('loadingIndicator');
const endMessage = document.getElementById('endMessage');

function loadMoreItems() {
    if (isLoading) return;
    isLoading = true;
    loadingIndicator.classList.add('visible');
    
    // Simulate API delay
    setTimeout(() => {
        const newItems = generateItems(currentPage, 10);
        newItems.forEach(item => {
            const itemEl = createItemElement(item);
            itemList.appendChild(itemEl);
        });
        
        currentPage++;
        isLoading = false;
        loadingIndicator.classList.remove('visible');
        
        // Stop after 5 pages
        if (currentPage > 5) {
            endMessage.style.display = 'block';
        }
    }, 500);
}

function createItemElement(item) {
    const div = document.createElement('div');
    div.className = 'item';
    div.innerHTML = `
        <div class="item-number">${item.id}</div>
        <div class="item-content">
            <h3 class="item-title">${item.title}</h3>
            <p class="item-desc">${item.description}</p>
        </div>
        <div class="item-badge">${item.tag}</div>
    `;
    return div;
}

scrollContainer.addEventListener('scroll', () => {
    if (scrollContainer.scrollTop + scrollContainer.clientHeight 
        >= scrollContainer.scrollHeight - 100) {
        loadMoreItems();
    }
});

// Load initial items
loadMoreItems();

Features

  • Automatic Loading - Loads content when user reaches bottom
  • Loading Indicator - Shows spinner during data fetch
  • Smooth Animation - New items slide in smoothly
  • End Detection - Shows message when no more items
  • Debounced Loading - Prevents multiple simultaneous requests
  • Threshold Control - Customize when loading starts

Use Cases

  • Social Media Feed - Twitter, Instagram, Facebook feeds
  • Image Gallery - Pinterest, Flickr, photo galleries
  • Product Listings - E-commerce product browsing
  • News Feed - Blog posts, articles, news streams
  • Search Results - Google, search engines pagination
  • Activity Stream - User activity logs and timelines

Accessibility

  • Use semantic HTML for item structure
  • Provide clear loading indicators for screen readers
  • Use aria-label on the loading spinner
  • Support keyboard navigation (Tab through items)
  • Announce when new items are loaded (ARIA live regions)
  • Provide alternative pagination option

Best Practices

  • Loading Threshold - Start loading 100-200px before bottom
  • Request Debouncing - Prevent multiple simultaneous requests
  • Visual Feedback - Always show loading spinner
  • Error Handling - Show error message if load fails
  • End Detection - Clearly indicate when no more items
  • Performance - Consider virtualizing large lists
