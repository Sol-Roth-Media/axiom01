Autocomplete

Autocomplete input with intelligent suggestions.

Basic Usage

Lightweight combobox behavior with keyboard navigation and ARIA states.

    Required Markup Contract

    • Use data-component="autocomplete" on the root element.
    • Include one input and one role="listbox" element for options.
    • Provide options via the data-options comma-separated attribute.
