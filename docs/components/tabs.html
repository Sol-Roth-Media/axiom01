Tabs

Tabs are used to organize and allow navigation between groups of content that are related and at the same level of hierarchy.

Accessibility

Axiom's tab component script handles all the necessary ARIA attributes and keyboard interactions to be fully accessible.

  • `role="tablist"`, `role="tab"`, and `role="tabpanel"` are used to define the structure for assistive technologies.
  • `aria-selected` indicates the currently active tab.
  • `aria-controls` links each tab to its corresponding tab panel.
  • Users can navigate between tabs using the arrow keys.
