Alert

Axiom01's alert component displays important messages with color-coded severity levels. Essential for system feedback and user guidance.

Live Examples

ℹ️

Information

This is an informational alert with important details for the user.

✓

Success

Your action has been completed successfully.

⚠️

Warning

Please be careful with this action as it may have consequences.

✕

Error

An error occurred. Please try again or contact support.

HTML Structure

<div class="alert alert-info">
    <div class="alert-icon">ℹ️</div>
    <div class="alert-content">
        <p class="alert-title">Title</p>
        <p class="alert-message">Message</p>
    </div>
</div>

Features

  • Multiple Types - Info, Success, Warning, Error
  • Color Coded - Visual severity indication
  • Icons - Type-specific icons
  • Left Border - Accent line for emphasis
  • Flexible Content - Title and message areas
  • Dismissible - Optional close button

Use Cases

  • System Messages - Status updates
  • User Feedback - Action results
  • Warnings - Important notices
  • Errors - Problem indicators
  • Information - Contextual help
  • Success - Completion confirmation

Accessibility

  • Use role="alert" for urgent messages
  • Color + icon for type indication
  • Clear, readable text
  • Sufficient contrast (WCAG AA)
  • Avoid relying on color alone
  • Announce to screen readers

Best Practices

  • Clear Messages - Be specific and actionable
  • Appropriate Type - Match severity to type
  • Visual Hierarchy - Use size and color strategically
  • Timely Display - Show when relevant
  • Icon Usage - Reinforce type with icons
  • Dismissible - Allow users to close if appropriate
