Playlist

Organize and display media playlists.

Features

  • Display and manage track queues
  • Drag-and-drop reordering
  • Shuffle mode for randomized playback
  • Built-in search to filter tracks
  • Add and remove items dynamically
  • Track duration display
  • Next/Previous navigation
  • Visual feedback for currently playing track

Data Format

Each track object in the playlist should have:

Property Type Required Description
id Number/String Yes Unique track identifier
title String Yes Track title
artist String Yes Artist or creator name
duration Number No Duration in seconds

JavaScript API

The playlist component provides a comprehensive API:

  • addItem(item) - Add track to playlist
  • removeItem(index) - Remove track by index
  • setItems(items) - Replace all tracks
  • getItems() - Get all tracks
  • clear() - Clear entire playlist
  • playNext() - Move to next track
  • playPrevious() - Move to previous track
  • getCurrentItem() - Get current track

Events

Event Detail Description
playlist-item-selected { item, index } User clicked a track
playlist-updated { items } Playlist contents changed
shuffle-toggled { shuffled: boolean } Shuffle mode toggled
add-item-requested {} Add button clicked

Integration with Audio Player

Connect playlist to an audio player:

  • Listen to 'playlist-item-selected' events
  • Call audio player's setSource() method
  • Update playlist UI when audio playback changes
  • Implement next/previous buttons using playNext/playPrevious

Best Practices

  • Track Duration: Always provide accurate duration for progress display
  • Unique IDs: Use consistent, unique identifiers for tracks
  • Performance: Virtual scrolling recommended for playlists with 500+ tracks
  • UI Feedback: Highlight currently playing track with visual indicator
