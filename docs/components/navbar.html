Accessibility Features

  • Semantic Navigation: Proper nav element with ARIA labels
  • Keyboard Navigation: Full keyboard support for all links and dropdowns
  • Screen Reader Support: Clear navigation landmarks and current page indication
  • Mobile Menu: Accessible mobile toggle with proper ARIA states
  • Focus Management: Logical tab order and visible focus indicators
