File Upload

Axiom01's file upload component provides an intuitive interface for selecting and uploading files. It supports drag-and-drop, file validation, and progress indication for a user-friendly experience.

HTML Structure

<!-- Drag and drop zone -->
<div class="upload-zone" role="button" tabindex="0" aria-label="Upload area">
    <div class="upload-icon">
        <span class="axicon render" data-name="Upload"></span>
    </div>
    <p class="upload-text">
        <strong>Click to browse</strong> or drag files here
    </p>
</div>

<input type="file" id="fileInput" class="hidden-input" multiple>
<div class="file-list" id="fileList"></div>

<!-- Simple file input -->
<label for="file">Upload File</label>
<input type="file" id="file" accept=".pdf,.doc">

JavaScript Implementation

const uploadZone = document.getElementById('uploadZone');
const fileInput = document.getElementById('fileInput');
const fileList = document.getElementById('fileList');

// Click to upload
uploadZone.addEventListener('click', () => fileInput.click());
uploadZone.addEventListener('keydown', (e) => {
    if (e.key === 'Enter' || e.key === ' ') {
        e.preventDefault();
        fileInput.click();
    }
});

// Drag and drop
uploadZone.addEventListener('dragover', (e) => {
    e.preventDefault();
    uploadZone.classList.add('drag-over');
});

uploadZone.addEventListener('dragleave', () => {
    uploadZone.classList.remove('drag-over');
});

uploadZone.addEventListener('drop', (e) => {
    e.preventDefault();
    uploadZone.classList.remove('drag-over');
    const files = e.dataTransfer.files;
    handleFiles(files);
});

// File input change
fileInput.addEventListener('change', (e) => {
    handleFiles(e.target.files);
});

function handleFiles(files) {
    Array.from(files).forEach(file => {
        const item = createFileItem(file);
        fileList.appendChild(item);
    });
    
    if (files.length > 0) {
        fileList.style.display = 'block';
    }
}

Features

  • Drag and Drop - Drop files directly onto the upload area
  • Click to Browse - Traditional file picker dialog
  • Multiple Files - Upload multiple files at once
  • File Validation - Accept specific file types
  • File Preview - Show uploaded files with metadata
  • Remove Option - Remove files before submission

Use Cases

  • Document Upload - Resume, invoice, or document submission
  • Image Gallery - Photo uploads for profiles or products
  • File Management - Cloud storage or file sharing
  • Form Attachments - Additional information submission
  • Batch Import - CSV/Excel data import
  • Media Upload - Video or audio file uploads

Accessibility

  • Use role="button" and tabindex="0" on upload zones
  • Include keyboard support (Enter/Space to open picker)
  • Provide clear file type restrictions
  • Use aria-label to describe upload areas
  • Show file validation error messages clearly
  • Allow removing files with keyboard

Best Practices

  • File Size Limit - Clearly display maximum file size
  • File Type Validation - Restrict to expected formats
  • Virus Scanning - Scan uploaded files server-side
  • Progress Indication - Show upload progress
  • Error Handling - Clear, actionable error messages
  • Drag Feedback - Visual feedback during drag-over
