File Upload Component

A drag-and-drop file upload component with progress indication, file validation, and accessibility features.

Basic File Upload

Drag-and-drop file upload with multiple file support and validation.

<div class="upload" data-component="file-upload">
  <label for="file-input">
    <input type="file" id="file-input" accept=".pdf,.doc,.docx" multiple hidden>
    <div>
      <i class="fas fa-cloud-upload-alt" aria-hidden="true"></i>
      <h4>Drop files here or click to upload</h4>
      <p>Supports PDF, DOC, and DOCX files up to 10MB</p>
    </div>
  </label>
  <div class="files" id="file-list" aria-live="polite"></div>
</div>

Accessibility Features

  • Keyboard Navigation: Full keyboard support for file selection
  • Screen Reader Support: Proper labels and ARIA attributes
  • Live Regions: Real-time upload status announcements
  • Focus Management: Clear focus indicators and logical tab order
  • Error Handling: Clear validation messages and recovery options
