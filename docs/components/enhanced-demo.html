JavaScript API

Manual Initialization

// After DOM ready
Axiom.initQueue(element)
Axiom.initSegmentedControl(element)
Axiom.initChipInput(element)
Axiom.initBacklit(element)

Event Listeners

// Segmented Control change
element.addEventListener(
  'segmented-change',
  (e) => console.log(e.detail.value)
)

Component Tracking

// Check initialized
Axiom.initializedComponents.has(el)

// Get instance
Axiom.initializedComponents
  .get(el).instance

Cleanup

// Remove single
Axiom.destroyComponent(element)

// Remove all
Axiom.destroyAllComponents()

Keyboard Shortcuts

Component Key Action
Segmented Control Tab Move between buttons
Segmented Control ← / → Navigate left/right
Segmented Control ↑ / ↓ Navigate up/down
Chip Input Enter / , Add chip
Chip Input Backspace Remove last chip (when empty)
Queue Drag Reorder items

Browser Console Test

Try these commands in the browser console to test the JavaScript API:

// Get all initialized components
console.log(Axiom.initializedComponents)

// Destroy all and reinitialize
Axiom.destroyAllComponents()
Axiom.init()

// Manually initialize a specific component
const queueEl = document.querySelector('.queue')
Axiom.initQueue(queueEl)
