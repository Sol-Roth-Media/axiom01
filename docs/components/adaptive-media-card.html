Features

  • Semantic HTML - Uses proper semantic tags (figure, header, main, footer)
  • Single Class - Just add `.adaptive-media` to any card
  • Responsive - Automatically adjusts layout based on viewport
  • Media-First - Prioritizes image/video for best viewing
  • Dark Mode - Full dark mode support
  • No JavaScript - Pure CSS solution
