HTML Structure

Pagination uses semantic <nav> with an unordered list and anchor links:

<nav aria-label="Page navigation">
    <ul class="pagination">
        <li><a href="#">Previous</a></li>
        <li><a href="#">1</a></li>
        <li><a href="#" class="active">2</a></li>
        <li><a href="#">3</a></li>
        <li><a href="#">Next</a></li>
    </ul>
</nav>

Styling

The .pagination class applies flexbox layout to arrange items horizontally:

.pagination {
    display: flex;
    flex-wrap: wrap;
    gap: var(--a-space-m);
    list-style: none;
    margin: 0;
    padding: 0;
}

.pagination a {
    display: inline-flex;
    align-items: center;
    justify-content: center;
    min-width: 40px;
    padding: var(--a-space-s) var(--a-space-m);
    border: 1px solid var(--ax-border);
    border-radius: var(--a-border-radius-md);
    background: var(--ax-background-secondary);
    color: var(--a-color-text);
    text-decoration: none;
    transition: all var(--a-transition-base) var(--a-ease-out);
}

.pagination a:hover {
    background: var(--a-color-primary-container);
    border-color: var(--a-color-primary);
    color: var(--a-color-primary);
}

.pagination a.active {
    background: var(--a-color-primary);
    color: white;
    border-color: var(--a-color-primary);
}

Use Cases

  • Search Results - Paginate through search results or listings
  • Blog Archives - Navigate between blog post listings
  • Product Listings - E-commerce product catalog pagination
  • Data Tables - Paginate large datasets
  • Document Galleries - Navigate through image or document collections

Accessibility

  • Use semantic <nav> element with aria-label
  • Active page indicator with .active class
  • Consider adding aria-current="page" to active link
  • All links must be keyboard focusable and operable
  • Provide visual and semantic indication of current page

Best Practices

  • Always show current page indicator
  • Disable or hide Previous/Next buttons when at start/end
  • Show page numbers for better navigation context
  • Keep pagination controls consistent in placement
  • Use clear labels like "Previous" and "Next"
  • Consider showing total number of pages
