Pagination Component

Navigation controls for splitting content across multiple pages with accessibility and keyboard support.

Basic Pagination

Complete pagination with previous/next links and numbered pages.

  • Previous
  • 1
  • 2
  • 3
  • …
  • 10
  • Next 
<nav class="pagination" role="navigation" aria-label="Pagination Navigation">
  <ul>
    <li>
      <a href="?page=1" aria-label="Go to previous page" aria-disabled="true">
        <i class="fas fa-chevron-left" aria-hidden="true"></i>
        Previous
      </a>
    </li>
    <li>
      <a href="?page=1" class="active" aria-current="page">1</a>
    </li>
    <li>
      <a href="?page=2">2</a>
    </li>
    <li>
      <span class="ellipsis" aria-hidden="true">…</span>
    </li>
  </ul>
</nav>

Simple Pagination

Minimal pagination with just previous/next controls.

Previous Page 2 of 10 Next 
<nav class="pagination simple">
  <a href="?page=1" class="prev" aria-label="Go to previous page">
    <i class="fas fa-chevron-left" aria-hidden="true"></i>
    Previous
  </a>
  <span class="info">Page 2 of 10</span>
  <a href="?page=3" class="next" aria-label="Go to next page">
    Next
    <i class="fas fa-chevron-right" aria-hidden="true"></i>
  </a>
</nav>

Accessibility Features

  • ARIA Navigation: Proper navigation landmark with descriptive labels
  • Current Page: aria-current="page" for screen reader identification
  • Keyboard Support: Full keyboard navigation with proper focus indicators
  • Descriptive Links: Clear ARIA labels for navigation context
  • Disabled States: Proper handling of first/last page boundaries
