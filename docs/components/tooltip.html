Tooltip Component

Axiom01's tooltip component provides a small, contextual popup that displays informative text when a user hovers over or focuses on an element. It's designed to be non-intrusive and enhance user understanding without cluttering the interface.

HTML Structure

A tooltip consists of a trigger element and the tooltip content itself, typically wrapped in a container.

<div class="tooltip" data-component="tooltip" data-placement="top">
		 <button class="button" data-tone="primary" data-tooltip-trigger aria-describedby="my-tooltip-id">Hover for Info</button>
		 <div role="tooltip" id="my-tooltip-id" data-tooltip-content aria-hidden="true">
				 This is a helpful tooltip!
		 </div>
 </div>

Styling

The .tooltip class provides the positioning context. Use data-tooltip-content for the hidden content node, data-tooltip-trigger on the trigger element, and data-placement on the container to control orientation.

  • .tooltip: Container for the trigger and content, providing relative positioning.
  • data-tooltip-trigger: The element that activates the tooltip (e.g., a button, icon, or text).
  • data-tooltip-content: The actual tooltip text, initially hidden and styled with background, text color, padding, and border-radius.
  • data-placement values (top, right, bottom, left) on the .tooltip container adjust the absolute position of the tooltip content.

JavaScript Integration

Tooltips use Axiom01's built-in component loader. Add data-component="tooltip" on each tooltip container and include js/axiom.js on the page.

<div class="tooltip" data-component="tooltip" data-placement="top">
  <button data-tooltip-trigger aria-describedby="tip-id">Top</button>
  <div role="tooltip" id="tip-id" data-tooltip-content aria-hidden="true">
    Info on top
  </div>
</div>

<script src="../../js/axiom.js"></script>

Accessibility

Tooltips are designed with accessibility in mind:

  • The trigger element has aria-describedby pointing to the tooltip's ID, linking the two for screen readers.
  • The tooltip content has role="tooltip".
  • Tooltips are activated on both hover (mouse users) and focus (keyboard users).
  • The content is hidden from assistive technologies when not visible.

Customization

Customize the appearance using Axiom01's CSS variables for colors, spacing, and typography. You can adjust the background, text color, padding, border-radius, and shadow of the tooltip content.
