Tooltip

Axiom01's tooltip component displays contextual information on hover. Supports multiple positioning options and smooth animations for enhanced user guidance.

Live Examples

Tooltip Positions

HTML Structure

<button class="tooltip-trigger">
    Hover me
    <div class="tooltip">
        This is helpful text
    </div>
</button>

<!-- Positioned variants -->
<button class="tooltip-trigger">
    Right tooltip
    <div class="tooltip right">Content</div>
</button>

<button class="tooltip-trigger">
    Bottom tooltip
    <div class="tooltip bottom">Content</div>
</button>

Features

  • Hover Trigger - Show on mouse over
  • Multiple Positions - Top, right, bottom, left
  • Arrow Pointer - Visual connection to trigger
  • Smooth Animation - Fade and slide effects
  • Auto Position - Avoid viewport edges
  • Dark Background - High contrast text

Use Cases

  • Help Text - Explain functionality
  • Keyboard Shortcuts - Show shortcut keys
  • Icon Clarification - Explain icon meaning
  • Abbreviations - Expand shortened text
  • Hints - Contextual guidance
  • Status Info - Additional context

Accessibility

  • Use aria-label for trigger elements
  • Include tooltip content in accessible tree
  • Support keyboard focus with Tab key
  • Announce tooltip to screen readers
  • High contrast colors (WCAG AA minimum)
  • Avoid moving essential content into tooltips

Best Practices

  • Concise Text - Keep tooltips brief
  • Clear Language - Avoid jargon
  • Smart Positioning - Stay within viewport
  • Delay - Don't show instantly
  • High Contrast - Ensure readability
  • Icon Support - Work with icon buttons
