Notification

Alert users to important information with semantic notifications. Supports multiple types: success, error, warning, and info.

Markup

Notifications use semantic HTML with ARIA roles and utility classes for styling:

<div class="notification success" role="alert">
    <span class="axicon render" data-name="CheckCircle"></span>
    <div>
        <strong>Success!</strong> Your action completed.
    </div>
    <button class="notification-close" aria-label="Close">&times;</button>
</div>

<!-- Types: success, error, warning, info -->

Types

  • success - Green, for positive feedback
  • error - Red, for errors or failures
  • warning - Orange, for warnings or caution
  • info - Blue, for informational messages

Features

  • Semantic HTML - Uses <div> with role="alert"
  • Icon Support - Include icons for visual clarity
  • Close Button - Allow users to dismiss
  • Type Variants - Four semantic notification types
  • Token-Based Colors - Uses design tokens for styling
  • Accessibility - ARIA roles for screen readers

Accessibility

  • role="alert" - Announces notification to screen readers
  • aria-label - Labels close button for context
  • Color + Icon - Never rely on color alone
  • Focus Management - Close button is keyboard accessible
  • Clear Text - Use descriptive messages

Best Practices

  • Keep messages concise and actionable
  • Use appropriate type for the message content
  • Include icons to reinforce the message type
  • Allow users to dismiss notifications
  • Position consistently on the page
  • Use color with icon/text, not color alone
