Notification Component

Toast notifications and alert messages for user feedback, system status, and important announcements.

Basic Notification

Standard notification message with icon and close button.

Information

Your changes have been saved successfully.

<div class="notification" role="alert" aria-live="polite">
  <div>
    <i class="fas fa-info-circle" aria-hidden="true"></i>
    <div>
      <h4>Information</h4>
      <p>Your changes have been saved successfully.</p>
    </div>
  </div>
  <button class="close" aria-label="Close notification">
    <i class="fas fa-times" aria-hidden="true"></i>
  </button>
</div>

Accessibility Features

  • Live Regions: Proper ARIA live regions for screen reader announcements
  • Alert Roles: Semantic alert roles for different notification types
  • Keyboard Support: Close button accessible via keyboard
  • Clear Messaging: Descriptive titles and actionable content
  • Auto-dismiss: Optional automatic dismissal with user control
