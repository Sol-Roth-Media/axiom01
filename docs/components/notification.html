Notification

Axiom01's notification component displays contextual alerts and messages to users. Supports multiple types (success, error, warning, info) with dismissible options and animations.

HTML Structure

<div class="notification success">
    <div class="notification-icon"><span class="axicon render" data-name="Check-Circle"></span></div>
    <div class="notification-content">
        <p class="notification-title">Success!</p>
        <p class="notification-message">Message text</p>
    </div>
    <button class="notification-close" aria-label="Close">×</button>
</div>

Features

  • Multiple Types - Success, error, warning, info
  • Icons - Visual type identification
  • Dismissible - Close button to remove
  • Action Buttons - Call-to-action within notification
  • Animations - Slide-in and fade effects
  • Auto-dismiss - Optional timeout feature

Use Cases

  • Form Submission - Success or error feedback
  • System Status - Alert notifications
  • User Actions - Confirmation messages
  • Warnings - Important notices
  • Updates - Software or content updates
  • Help Text - Contextual information

Accessibility

  • Use role="alert" for urgent messages
  • Include descriptive title and message
  • Color + icon for type indication
  • Include close button with aria-label
  • Announce to screen readers on appearance
  • Allow keyboard dismissal

Best Practices

  • Clear Messages - Be specific and actionable
  • Appropriate Type - Match severity to type
  • Color + Icon - Redundant visual cues
  • Auto-Dismiss - Use sparingly, not for errors
  • Position - Typically top-right or center
  • Timing - Don't hide before user reads
