Button Component

Use Axiom's custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Examples

<button class="button">Default</button>
<button class="button primary">Primary</button>
<button class="button secondary">Secondary</button>
<button class="button success">Success</button>
<button class="button warning">Warning</button>
<button class="button error">Error</button>

Button Sizes

Fancy larger or smaller buttons? Add `.large` or `.small` for additional sizes.

<button class="button small">Small</button>
<button class="button">Default</button>
<button class="button large">Large</button>

Buttons with Icons

You can add icons to your buttons for additional visual context.

<button class="button primary"><i class="fas fa-plus"></i> Create</button>
<button class="button secondary"><i class="fas fa-download"></i> Download</button>
<button class="button error"><i class="fas fa-trash"></i> Delete</button>

Button States

Buttons can be in a disabled state. Add the `disabled` boolean attribute to any ` 

<button class="button primary" disabled>Disabled</button>
<button class="button secondary" disabled>Disabled</button>
