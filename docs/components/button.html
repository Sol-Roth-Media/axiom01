Button Component

Axiom01's button component provides clear, semantic, and accessible calls to action. Buttons are designed to be intuitive and highly customizable through CSS variables, ensuring they fit seamlessly into any theme.

Default Buttons

<button class="button">Default Button</button>
	<button class="button" data-tone="primary">Primary Button</button>
	<button class="button" data-tone="secondary">Secondary Button</button>

Button Sizes

<button class="button" data-tone="primary" data-size="small">Small Button</button>
	<button class="button" data-tone="primary">Base Button</button>
	<button class="button" data-tone="primary" data-size="large">Large Button</button>

Outline Buttons

<button class="button" data-tone="primary" data-variant="outline">Primary Outline</button>
	<button class="button" data-tone="secondary" data-variant="outline">Secondary Outline</button>
	<button class="button" data-variant="outline">Default Outline</button>

Buttons with Icons

<button class="button" data-tone="primary"><i class="fas fa-plus"></i> Add Item</button>
	<button class="button" data-tone="secondary"><i class="fas fa-save"></i> Save</button>
	<button class="button"><i class="fas fa-trash"></i> Delete</button>

Disabled Buttons

<button class="button" data-tone="primary" disabled>Disabled Primary</button>
	<button class="button" data-tone="secondary" disabled>Disabled Secondary</button>
	<button class="button" disabled>Disabled Default</button>

Link Buttons

Use <a> tags with .button class for navigation.
