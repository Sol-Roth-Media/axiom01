Typography Reference

Every text pattern available in Axiom01 — fully automatic, zero classes needed.

Headings

Headings scale mathematically using --a-heading-scale: 1.15. All sizes are calculated — change one variable to adjust the entire scale.

<h2>Heading 2 — Section Title</h2>

<h4>Heading 4 — Card Title</h4>
<h5>Heading 5 — Label</h5>

Customising the Scale

The entire heading scale is derived from two variables. Change them globally:

:root {
  /* Base font size — all headings scale from this */
  --a-font-size-base: 1rem;

  /* Scale multiplier — each heading level is 1.15× the one below */
  --a-heading-scale: 1.15;

  /* Heading margins */
  --a-heading-margin-top: var(--a-margin-medium);
  --a-heading-margin-bottom: var(--a-margin-base);

  /* Reading width for paragraphs */
  --ax-reading-width: 65ch;
}
