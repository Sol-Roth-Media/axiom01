External Integrations & CMS Themes

Axiom01 is designed to be framework-agnostic and can be integrated into any web project. This section provides guidance and resources for integrating Axiom01 with popular Content Management Systems (CMS) and other external platforms.

Drupal 11 Theme (Coming Soon)

We are actively developing a dedicated Drupal 11 theme that fully leverages the Axiom01 framework. This theme aims to provide a seamless and beautiful experience for Drupal users, ensuring that all core Drupal components and pages are styled according to Axiom01's semantic-first philosophy.

Key features of the upcoming Drupal 11 theme:

  • Comprehensive Theming: Templates for core Drupal components including Views, contextual editing links, nodes (pages), and user profiles.
  • Axiom01 Component Integration: Replaces default Drupal UI elements (e.g., accordions) with their Axiom01 counterparts.
  • Semantic Structure: Designed to minimize extra classes, adhering to Axiom01's philosophy of clean, semantic HTML.
  • Header & Navigation: Fully functional header with site name and main menu system.
  • Backward Compatibility: Future Axiom01 releases will prioritize backward compatibility to ensure stability for the Drupal theme.

Stay tuned for updates on its development and release!

Other Integrations

Axiom01's CSS-variable-driven architecture makes it highly adaptable to various frameworks and libraries. We encourage the community to develop and share integrations with other popular platforms.

If you're interested in contributing to an integration or have developed one yourself, please refer to our Contributing Guide.
