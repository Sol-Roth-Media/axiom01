External Integrations & CMS Themes

Axiom01 is designed to be framework-agnostic and can be integrated into any web project. This section provides guidance and resources for integrating Axiom01 with popular Content Management Systems (CMS) and other external platforms.

Drupal 11 Starter Theme (Bootstrap Available)

A scaffold for the dedicated axiom01-drupal11-starter-theme repository now exists in this repository and is ready to be extracted into its own repo.

Scaffold location:

/tmp/workspace/Sol-Roth-Media/axiom01/starter-themes/axiom01-drupal11-starter-theme/

Included starter coverage:

  • Theme scaffolding: Drupal 11 info, libraries, and theme hooks.
  • Menus: Main, footer, and utility menu Twig templates.
  • Core templates: Page, node, views, forms, and status messages.
  • Component layer: Reusable Twig partials mapped to Axiom01 semantics.
  • Governance docs: Compatibility contract, upgrade guide, and cross-repo validation checklist.

For maintainability, only the Axiom01 core-supported surface is guaranteed stable for Drupal integration.

Other Integrations

Axiom01's CSS-variable-driven architecture makes it highly adaptable to various frameworks and libraries. We encourage the community to develop and share integrations with other popular platforms.

If you're interested in contributing to an integration or have developed one yourself, please refer to our Contributing Guide.
