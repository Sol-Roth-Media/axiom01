Integrations

Axiom01 is designed to drop into real projects with minimal ceremony: static sites, documentation portals, CMS themes, and progressively enhanced interfaces.

Static HTML

Use semantic HTML plus the compiled CSS and optional JavaScript helpers. This is the simplest and most framework-native setup.

Design Systems

Adopt the token layer first, then add component classes where a semantic wrapper improves clarity and consistency.

CMS / Templates

Keep generated markup semantic, avoid utility drift, and map theme templates to the framework’s existing component roots.
