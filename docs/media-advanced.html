Media & Icons

Axiom01 includes a comprehensive icon system and media best practices for images, videos, and other visual content.

Axicons - The Icon System

Axiom01 includes 204 built-in icons called Axicons. These are native SVG icons that are lightweight, scalable, and accessible. No external dependencies or font files needed.

→ Browse all Axicons in the icon browser

Why Axicons?

  • Native SVG - No font fallbacks or rendering issues
  • Lightweight - Only ~15 KB for the entire library
  • Scalable - Perfect at any size from 16px to 256px
  • Accessible - Built with aria-hidden and semantic structure
  • Customizable - Control color via CSS, no image replacement needed
  • Consistent Design - All icons follow the same visual style

Using Axicons

There are two ways to use axicons in your projects:

1. Using the Axicon Renderer (Recommended)

Use the axicon render class to automatically render icons from the JavaScript library:

<span class="axicon render" data-name="Check-Circle"></span>
<span class="axicon render" data-name="Search"></span>
<span class="axicon render" data-name="Menu"></span>

2. Inline SVG

For cases where you need direct control, embed the SVG directly:

<svg viewBox="0 0 24 24" width="24" height="24" aria-hidden="true">
  <circle cx="12" cy="12" r="10"></circle>
  <line x1="12" y1="16" x2="12" y2="12"></line>
  <line x1="12" y1="8" x2="12.01" y2="8"></line>
</svg>

Sizing Icons

<!-- Small icon (16px) -->
<span class="axicon render" data-name="Check" style="font-size: 16px;"></span>

<!-- Default (24px) -->
<span class="axicon render" data-name="Check"></span>

<!-- Large icon (32px) -->
<span class="axicon render" data-name="Check" style="font-size: 32px;"></span>

Coloring Icons

<!-- Inherit text color -->
<span class="axicon render" data-name="Check"></span>

<!-- With specific color -->
<span class="axicon render success" data-name="Check-Circle"></span>

<!-- Custom color -->
<span class="axicon render" data-name="Warning" style="color: var(--a-color-warning);"></span>

Icon Best Practices

  • Always provide context - Pair icons with text labels, especially for interactive elements
  • Use aria-hidden="true" for decorative icons - Screen readers skip purely decorative icons
  • Consistent sizing - Stick to standard sizes (16px, 24px, 32px, 48px) for visual consistency
  • Color contrast - Ensure icons have sufficient contrast against the background (4.5:1 minimum)
  • Active states - Use different colors or fills to indicate active/selected states
  • Accessible icons - For functional icons, include proper aria-labels or title attributes

Images & Media

Best practices for using images, videos, and other media in your Axiom01 projects.

Responsive Images

<!-- Simple responsive image -->
<img 
  src="image.jpg" 
  alt="Descriptive text"
  style="width: 100%; height: auto; border-radius: var(--a-border-radius-lg);"
>

<!-- Picture element for art direction -->
<picture>
  <source media="(min-width: 900px)" srcset="image-large.jpg">
  <source media="(min-width: 600px)" srcset="image-medium.jpg">
  <img src="image-small.jpg" alt="Descriptive text">
</picture>

<!-- With srcset for different densities -->
<img 
  src="image.jpg"
  srcset="image@2x.jpg 2x, image@3x.jpg 3x"
  alt="Descriptive text"
>

Image in Card

<article class="card">
  <img src="thumbnail.jpg" alt="Card image" style="width: 100%; border-radius: var(--a-border-radius-lg) var(--a-border-radius-lg) 0 0;">
  <div class="component-section">
    <h3>Card Title</h3>
    <p>Card description...</p>
  </div>
</article>

Embedded Video

<!-- Responsive iframe -->
<div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden;">
  <iframe 
    style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"
    src="https://www.youtube.com/embed/dQw4w9WgXcQ"
    title="YouTube video player"
    frameborder="0"
    allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
    allowfullscreen
  ></iframe>
</div>

SVG Graphics

<!-- Inline SVG -->
<svg viewBox="0 0 100 100" style="width: 100px; height: 100px;">
  <circle cx="50" cy="50" r="40" fill="var(--a-color-primary)"></circle>
</svg>

<!-- External SVG -->
<img src="graphic.svg" alt="Graphic description">

Accessibility for Media

Ensure all media is accessible to all users:

  • Alt Text - Always include descriptive alt text for images
  • Captions - Add captions for videos (especially audio-dependent content)
  • Transcripts - Provide transcripts for podcasts and important videos
  • Icon Labels - Use aria-labels or title attributes for icon-only buttons
  • Color Contrast - Ensure icons and graphics have proper contrast
  • Keyboard Navigation - All interactive media controls must be keyboard accessible

Performance Tips

Optimize media for faster page load times:

  • Image Optimization - Use modern formats (WebP) with fallbacks
  • Lazy Loading - Use the loading="lazy" attribute for below-the-fold images
  • SVG Optimization - Minify SVGs to reduce file size
  • Video Optimization - Use appropriate codecs and bitrates for streaming
  • Responsive Images - Serve different image sizes for different devices
