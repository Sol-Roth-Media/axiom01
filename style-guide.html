Axiom01 Style Guide & Aesthetic Theme

This guide demonstrates how to create visually consistent, beautiful, and responsive layouts using Axiom01. Learn to use semantic markup, variable-driven spacing, scalable typography, and advanced theming for a modern, aesthetic UI.

Content Flow & Layout Principles

Use semantic HTML elements (<header>, <section>, <article>, <footer>) to structure your content. Axiom01's grid and flex utilities, combined with variable-driven gaps, ensure even flow and spacing.

Use .card header, .card section, and .card footer for clean, maintainable styles.

Grid layouts use display: grid and gap: var(--a-grid-gutter) for even spacing. Flex layouts use gap and align-items for vertical rhythm.

Typography & Scaled Font Sizes

Typography is managed with variables for font size, weight, and line height. Scale headings and text responsively using var(--a-font-size-h1) to var(--a-font-size-small).

  • Font family: var(--a-font-family-sans) for body, var(--a-font-family-mono) for code.
  • Font weights: var(--a-font-weight-bold), var(--a-font-weight-medium), var(--a-font-weight-normal).
  • Line height: var(--a-line-height-base) for paragraphs, var(--a-line-height-heading) for headings.

Spacing: Padding, Margin, & Gaps

All spacing is managed with variables for consistency and easy theme switching. Use var(--a-padding-*), var(--a-margin-*), and var(--a-grid-gutter) for layout rhythm.

Large Padding
Small Padding
  • Use var(--a-padding-small), var(--a-padding-large) for inner spacing.
  • Use var(--a-margin-base), var(--a-margin-xlarge) for outer spacing.
  • Grid/flex gaps: var(--a-grid-gutter) for even columns/rows.

Advanced Theming: Beyond Colors

Axiom01 themes can change more than colors: update radii, shadows, gradients, surface effects, and spacing for a unique look. Example: the Aesthetic theme below.

Aesthetic Theme Demo

Gradients, large radii, soft shadows, and surface effects create a modern, aesthetic feel.

Surface Variant

Use var(--a-color-surface-variant) for cards, modals, and overlays.

  • Change border radius for softer or sharper corners.
  • Update shadows for depth and elevation.
  • Use gradients for backgrounds and accents.
  • Adjust surface colors for cards, modals, and overlays.
  • Scale spacing for a more open or compact feel.

Switch themes by changing data-theme="aesthetic" on <html>. All variables update automatically.
