Navbar Advanced Testing

Back to Home | Basic Navbar Docs

This page is dedicated to testing and showcasing advanced navbar features without affecting the existing navbar implementation. Use the controls below to toggle different features and experiment with new functionality.

Navbar Controls

Toggle different navbar features to see how they work:

Dropdown Menu Support

The advanced navbar includes dropdown menu support. Hover over the "Components" link in the navbar to see the dropdown menu.

Features:

  • Hover and focus-within activation
  • Keyboard navigation support
  • Proper ARIA attributes for accessibility
  • Mobile-responsive design

Accessibility Enhancements

The advanced navbar includes several accessibility improvements:

  • Skip to main content link (try pressing Tab when the page loads)
  • Enhanced keyboard navigation
  • ARIA live regions for menu state announcements
  • Improved focus states for all interactive elements

Scroll-Aware Behavior

When enabled, the navbar will change appearance based on scroll position:

  • Becomes more compact when scrolling down
  • Increases shadow depth for better visual hierarchy
  • Smooth transitions between states

Try enabling the "Scroll-Aware Behavior" option and scroll down to see the effect.

Search Functionality

The search input in the navbar demonstrates:

  • Expandable search field on focus
  • Proper keyboard accessibility
  • Responsive design for mobile devices

Note: This is a demonstration only and doesn't perform actual searches.

This area exists to provide enough content to test the scroll-aware navbar behavior. Scroll up and down to see how the navbar responds when the scroll-aware option is enabled.
