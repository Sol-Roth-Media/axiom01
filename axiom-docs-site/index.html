Axiom UI Framework

A minimal, semantic, and flexible CSS framework for building beautiful interfaces with less markup.

Get Started GitHub

Overview

Axiom is a lightweight, modern CSS framework designed to help you build beautiful interfaces with minimal markup. It focuses on semantic HTML, clean CSS, and a flexible design system that adapts to your needs.

Semantic HTML

Build with meaningful, accessible HTML elements instead of endless utility classes.

Minimal Classes

Use a single class for components and rely on semantic HTML for structure.

CSS Variables

Customize everything with CSS variables for colors, spacing, typography, and more.

Philosophy

Axiom follows a set of core principles that guide its design and implementation:

  • Semantic HTML First: Use the right HTML elements for the right job, enhancing accessibility and SEO.
  • Minimal Classes: Assign a single class for the main component and use semantic elements for structure.
  • Style with Element Selectors: Target semantic elements inside components in your CSS.
  • CSS Variables for Everything: Use variables for colors, spacing, typography, and more for easy customization.
  • Accessibility Built-in: All components are designed with accessibility in mind, including ARIA attributes and keyboard navigation.

Installation

You can include Axiom in your project using CDN, npm, or by downloading the files directly.

CDN

<link rel="stylesheet" href="https://cdn.example.com/axiom/axiom.min.css">

NPM

npm install axiom-ui

Download

Download the latest version from GitHub.

Quick Start

Get started with Axiom by including the CSS file in your HTML and using the semantic components.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Axiom Project</title>
  <link rel="stylesheet" href="https://cdn.example.com/axiom/axiom.min.css">
</head>
<body>
  <header>
    <nav class="navbar">
      <a href="#" class="logo">My Project</a>
      <ul class="navbar-menu">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
      </ul>
    </nav>
  </header>

  <main class="container">
    <section class="hero">
      <h1>Welcome to My Project</h1>
      <p>Built with Axiom UI Framework</p>
      <button class="button primary">Get Started</button>
    </section>
  </main>

  <footer>
    <p>© 2025 My Project</p>
  </footer>
</body>
</html>
