Installation & Setup

To use Axiom01, include the following CSS files in your <head>:

<link rel="stylesheet" href="css/axiom.css">
<link rel="stylesheet" href="css/navbar.css">
<link rel="stylesheet" href="css/site.css">

Optionally, add the Font Awesome CDN for icons:

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.2/css/all.min.css">

Build beautiful, accessible interfaces with less markup. Semantic-first, themeable, and fast.

Get Started Component Demos GitHub

Why Axiom01?

Axiom01 is designed for developers who want semantic, maintainable, and accessible UI with minimal class clutter. Unlike utility-first frameworks, Axiom01 leverages CSS variables and semantic HTML for easier customization and better accessibility.

Comparison:

Axiom01

  • Semantic HTML
  • CSS variables for theming
  • Accessible by default
  • Minimal class usage
  • Easy theme switching

Utility Frameworks

  • Heavy class usage
  • Harder to customize themes
  • Accessibility requires extra effort
  • Higher learning curve
  • More markup bloat

Quick Start

  1. Download or install Axiom01: 
    npm install axiom01
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/axiom01@latest/css/axiom.css">
<script src="https://cdn.jsdelivr.net/npm/axiom01@latest/js/axiom.js"></script>
  2. Include the CSS and JS in your HTML: 
    <link rel="stylesheet" href="css/axiom.css">
<script src="js/axiom.js"></script>
  3. Start using semantic HTML and Axiom components!

Hello World Example

<main class="container">
  <h1>Hello, Axiom01!</h1>
  <button class="btn primary">Click Me</button>
</main>

Troubleshooting

  • If styles do not appear, check your CSS/JS file paths.
  • Make sure your HTML <body> uses class="axiom01" for theme support.
  • Use the theme switcher in the navbar to test different themes.

Navbar Component

The navbar is a responsive, accessible navigation bar. It supports logo, links, theme switcher, and mobile menu toggle out of the box.

Example

<nav class="navbar" aria-label="Main Navigation">
  <a href="#" class="logo">[Ax01]</a>
  <button class="menu-toggle" aria-label="Toggle Navigation" aria-expanded="false" aria-controls="nav-links">
    <i class="fa-solid fa-bars"></i>
  </button>
  <ul class="nav-links" id="nav-links">
    <li><a href="#docs">Docs</a></li>
    <li><a href="#components">Components</a></li>
    <li><a href="#themes">Themes</a></li>
    <li><a href="#accessibility">Accessibility</a></li>
    <li><a href="#examples">Examples</a></li>
    <li><a href="https://github.com/Sol-Roth-Media/axiom01" target="_blank">GitHub</a></li>
  </ul>
  <div class="theme-switcher-container">
    <select id="theme-select">
      <option value="light">Light</option>
      <option value="dark">Dark</option>
      <option value="aesthetic">Aesthetic</option>
      <option value="cyberpunk">Cyberpunk</option>
    </select>
  </div>
</nav>

Features

  • Sticky positioning and responsive layout
  • Accessible ARIA attributes
  • Theme switcher for instant style changes
  • Mobile menu toggle with smooth animation

Grid System

Axiom01 includes a simple, flexible grid system for layouts. Use .row and .col classes for responsive columns.

Example

<div class="row">
  <div class="col">Column 1</div>
  <div class="col">Column 2</div>
  <div class="col">Column 3</div>
</div>

Features

  • Responsive columns stack on small screens
  • Customizable via CSS variables
  • Supports nested grids

Advanced Example

<div class="row">
  <div class="col col-2">2/12</div>
  <div class="col col-8">8/12</div>
  <div class="col col-2">2/12</div>
</div>

Use .col-N for custom column widths (N = 1-12).

Buttons

Axiom01 provides several button styles for different actions. Use btn and a modifier class for color or style.

Example

<button class="btn primary">Primary</button>
<button class="btn secondary">Secondary</button>
<button class="btn success">Success</button>
<button class="btn danger">Danger</button>
<button class="btn outline">Outline</button>

Features

  • Accessible focus states
  • Customizable via CSS variables
  • Works with <a> and <button> elements

Alerts

Use alerts to display important messages. Available types: success, error, info, warning.

Example

<div class="alert success">Success!</div>
<div class="alert error">Error!</div>
<div class="alert info">Info message.</div>
<div class="alert warning">Warning!</div>

Features

  • Distinct colors for each alert type
  • Optional icons (Font Awesome recommended)
  • Dismissible with custom JS

Theming & Customization

Axiom01 uses CSS variables for easy theming. Switch themes using the selector in the navbar, or override variables in your own CSS.

Example: Override Theme Colors

:root {
  --a-color-primary: #e91e63;
  --a-color-secondary: #009688;
}

Available Themes

  • Light
  • Dark
  • Aesthetic
  • Cyberpunk

Custom Theme Example

[data-theme="mytheme"] {
  --a-color-background: #222;
  --a-color-on-background: #fff;
  --a-color-primary: #ff9800;
}

Apply your theme by setting data-theme="mytheme" on <html> or <body>.
