How to Use: Breadcrumbs

<nav aria-label="Breadcrumb" class="breadcrumb">
  <ol>
    <li><a href="/">Home</a></li>
    <li aria-hidden="true">/</li>
    <li><a href="/components">Components</a></li>
    <li aria-hidden="true">/</li>
    <li><span>Current Page</span></li>
  </ol>
</nav>

How to Use: Pagination

<nav aria-label="Pagination" class="pagination">
  <ul>

</nav>

How to Use: Menus & Quick Links

<!-- Horizontal Menu -->
<nav class="menu">
  <ul>
    <li><a href="/">Home</a></li>
    ...
  </ul>
</nav>

<!-- Vertical Menu -->
<nav class="menu vertical">
  <ul>
    <li><a href="/">Home</a></li>
    ...
  </ul>
</nav>

<!-- Dropdown Menu -->
<div class="dropdown">
  <button aria-haspopup="true" aria-expanded="false">Menu</button>
  <ul>
    <li><a href="/">Home</a></li>
    ...
  </ul>
</div>

<!-- Jump Menu -->
<select onchange="if(this.value) location=this.value;">
  <option value="">Select...</option>
  <option value="/">Home</option>
  ...
</select>

Best Practices & Theming

  • Use semantic HTML for navigation (<nav>, <ul>, <li>, <a>).
  • Use CSS variables for colors, spacing, and radius.
  • Ensure keyboard and screen reader accessibility.
  • Menus and links should be easy to find and use.
