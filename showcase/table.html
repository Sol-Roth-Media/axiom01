Table Component

Overview

The Table component displays tabular data with semantic HTML. It is responsive, accessible, and uses minimal classes. Style only the table, th, td, and caption elements directly. Avoid BEML and compound classes.

Usage

<table>
  <caption>Project Status</caption>
  <thead>
    <tr>
      <th>Project</th>
      <th>Status</th>
      <th>Progress</th>
      <th>Owner</th>
      <th>Due Date</th>
      <th>Notes</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alpha</td>
      <td>In Progress</td>
      <td>60%</td>
      <td>Jane</td>
      <td>2025-08-01</td>
      <td>On track</td>
    </tr>
    <tr>
      <td>Beta</td>
      <td>Complete</td>
      <td>100%</td>
      <td>John</td>
      <td>2025-09-15</td>
      <td>Waiting for review</td>
    </tr>
  </tbody>
</table>

Accessibility & Usage Tips

  • Use semantic <table>, <thead>, <tbody>, <th>, <td>, and <caption> for structure.
  • Wrap tables in a div with overflow-x:auto for horizontal scrolling on small screens.
  • Use clear, descriptive column headers.
  • Provide a <caption> for accessibility.
  • Style only the semantic elements and use minimal classes.
